# 大狼

# 大狼

## 文/迦南

这天太阳快落山时，桑丘在自家的麦田里发现了一只大狼。它侧身瘫倒在那片祖传的庄稼地中央，鼻子里不时喷吐着热气，一动不动，得走近了方能听见它的低嚎。 再过十天，就该收割了，今年的长势好得不一般呀。桑丘看着大片被压坏的麦子，心疼了。那些伤想必令它疼得厉害，大概在这麦田里翻腾挣扎了许久，才丧尽了力气。桑丘这样想着，那略显粗糙的同情心从麦子转移到了大狼上。 细细打量过这只不幸的狼后，桑丘着实吃了一惊。这可真是只巨大的狼，它的毛比麦秆还要长，那尾巴足够像被子一样将人盖住，从微张的嘴里露的牙齿，与桑丘的小腿一般大。当它重重呼出的一口气吹在桑丘脸上时，他切实地感到了恐怖。 走吧，走吧。这不该我管。它太大了，我怎能帮得了它。即使它伤成这样，谁也无法保证它不会伤到我。 桑丘跳跃一般地向后退了几步，估摸着自己逃出了大狼爪子能够到的范围。该走了，桑丘在心里对自己说，却又不甘心将目光从大狼身上移开。正是这时，他注意到了大狼那微微张开的眼中，反射着夕阳的红光。那光很是微弱，却让人觉得不至于熄灭。 正是这一点反射来的光，让桑丘感到了心安。大狼低低轻嚎着，宛似一个农忙后招呼桑丘去饮烧酒的老友。一瞬间，桑丘竟难过得要流出泪来。这样大的身躯，要生存着，恐怕比自己还要不易吧。这样大的身躯，恐怕在同类中也是要感到孤独的吧。谁能忍心在它身上留下这样的伤啊。桑丘那略带傻气的善心突然像气球一样地膨胀了。 他下了要救活它的决心。 于是，他返身朝自己在麦田边上的小木屋走去。狼大概以为桑丘将要丢下它了，竟颇有些骄傲地发出威胁的叫喊，那声音，像是卡在喉咙中的一口痰发出的怪响。桑丘回望了它一眼，那眼中已没有悲戚的光芒，也没有被遗弃的哀怨，取而代之的是甚为尖锐的恨意。 桑丘从小木屋中抱来一堆干木，小心翼翼地靠近它的后腿。待他走近那狼时，便看见了其痛苦的所在。挨着根部的地方受了伤，一连串的口子延伸到腹下，筋肉暴露在外，并不时抽动。桑丘将干木置于地上，倒上些烈酒，从衣袋中掏出两片打火石，咔咔地敲打着。当那火星跌落在干木堆上时，远方的太阳恰在地平线后收起了最后一丝光。火焰腾起的瞬间，桑丘竟觉得太阳的光辉也不过如此。这突然出现的一团火让看似濒死的大狼受了不小的惊吓，它不住用未受伤的前足刨着身下的麦地，只想将自己推离火焰。 桑丘没有安慰狼的办法，好在它并无多余的力气，扑腾一阵也便消停了。桑丘细细看了看那些伤口，心里越发地感到奇怪。这些伤口杂乱无章，有深有浅地堆叠在一起，无法看出是何物所致。有些伤口尚有新鲜的血液流出，有些却在溢着脓。 需先消毒，否则更多的感染不能避免。桑丘这样想着，动手剪去伤口周边的长毛，将一块马蹄铁用火钳夹住，置于火上烤。待烤得略红时，桑丘将马蹄铁对准着伤口周边的皮肤按下去。大狼痛极了，再次发出威胁似的喊叫，但声音显然不如第一次那样骇人。 桑丘如是重复了几次，大狼的叫喊一次比一次无力。 忍一忍便过去了。桑丘竟不自觉地对狼说出了这样的话。 消过毒之后，他看上去似乎比狼受了更多折磨。 我确实不适合目睹这痛苦。他想。 他注意到狼的前爪已经深深刺入麦子下的泥中了。 再来便是繁琐冗长的过程了，桑丘用织毛衣所用的长针一点点将那些较深的伤口缝合。比起马蹄铁，长针所带来的痛苦还至于让狼产生太剧烈的反应。 而那些较浅的伤口，桑丘也用布为其涂上药水。 “大狼啊大狼。”这老实的农民边为其上药，一边碎碎念道：“传说若有野兽生得如你这般巨大，体内必藏有神灵。也有人说，神灵故意要扮作野兽来探人的善心。若好生待之，将得神的回报。” “我是真心愿救你，你若能活，我也不求什么。这一带总有农人被狼所伤，你生得这样大，是可以阻止这不幸的发生吧？” “我家里九代单传，全靠这一小片麦地过活，从没缩水过，也从没进取过。”桑丘看着大狼身下压着的麦子，又心疼了。 大狼的后腿微微抽动了一下，桑丘即又安慰自己道：“没事没事，这样的巨兽，可遇不可求，或许还要带来运气的。”说罢咧开嘴似乎是想笑，但那一口乱牙让这表情看起来毫无喜感。 大狼的喉咙里依旧发出怪响，在夜空下的麦田里显得尤为突兀。 这是某种表示吧。桑丘快活地想，勤快地拨掉一个伤口边发硬的泥土。 不远的地方，有马蹄声作响，渐渐近了。桑丘认得那是个本地的猎人。 “你见过这样大的狼没有？”桑丘远远地便对他喊道。 猎人到了火堆旁，翻身下马，向桑丘讨了些酒喝。桑丘显然是有些懊恼和失望，这猎人竟一点也未流露出惊奇的神色。 “你见过这样大的狼没有？”桑丘又问了遍。 “没见过。”猎人说罢呵呵笑了，像是看穿了桑丘的意图。 “狼都一样，不管是大狼或是狼崽。既然都一样，太大或太小又有什么可惊讶的。”猎人接着说。 “不过，倒是可以卖个好价钱。”他说罢对着大狼举起了枪。 桑丘一把将枪按下，他期盼已久的猎人吃惊的表情终于出现了。 “怎么，你不卖？” “不卖。” “留着何用？” “不留着，救活它。” “救它何用？” “你看它像是普通的狼吗？” “像。” 猎人的回答干净得彻底，桑丘厌烦了与他的对话，却总有些不甘的情绪作祟，促着他又开了口：“这样大的狼肯定不一般。救了它会得神佑的吧。” 猎人哈哈大笑，将还未吞下肚的酒喷得漫天飘散。 “或许不是神佑我一人的，譬如让此地丰收一类的，那么受益的也不仅是我呀。”桑丘赶忙补充道。 猎人摆了摆手，示意他不要再说下去了，他已笑得难受了。 待他缓过气来，只是轻松地说“那么好吧，若有神佑，我定也有份。”说罢费力地爬上马背，哼着来路不明的歌，一路远去了。 桑丘恨恨地继续为狼上药。这愤恨一半来自于自己的期待未被满足，另一半则是因为猎人对狼的态度。 他们都不愿意救这样的一只大狼么？他们竟要留着它慢慢淌血致死？ 桑丘多少有些惆怅，但是要救活狼的想法却不曾动摇。 你会明白的吧。他轻轻抚着大狼的长毛道。 每个人都有自己无法做到的事，于我而言，那便是无法让你死去。因为我正目睹着你的苦难。我从不让绝望侵袭自己，若任由你这样死去，不免要染上你死前的绝望。日后便不可不回想，如此，大概很久也无法逃脱其中。 至于神佑，那不过是一个貌似可以接受的托词而已。每个人都要找这样的托词，猎人也不例外。人总需要托词来支持他们那不被理解的行动。 农民桑丘这样安慰着自己和狼，直到夜色逐渐退去。 大狼没有任何好转的迹象。它的生命似乎正随时间一点一点地流走。 是需要吃的东西吧。肉体上的疼痛往往要消耗掉相当的体力。桑丘想起了这点后，便回小木屋，拎出两只活鸡。 “这恐怕不够你吃饱，但是应够你活下去。”他说着，走到大狼微张的嘴边，要将那活鸡放下。 正是这时，他方才看见了从狼的下颚歪歪扭扭延伸到脖子的一条巨大伤口，因为靠近地面麦子的缘故，一直未被发现。血已淌了一地，但却看不出停下的可能。 桑丘的心猛然被揪住了，痛恨起了自己。 这个伤口也需要消毒，缝合，这样才能救回它。 桑丘不再多想，转身回到火堆边，又开始烤那马蹄铁。 当他用火钳夹着马蹄铁走到那伤口边，打算最后完成这一切时，大狼迅速地将头一扭，拦腰将桑丘咬住，在空中来回甩了两下，他便断了。 断了的桑丘落在麦地上，心里所想的是尽快地爬远，因为大狼身边若是死了一个人，大概就不可能再有人愿来救它了。 只有上半截身子的桑丘爬了一米，突然想起自己的下半截身子还留在原地，便又艰难地在自己的散落的肠子中转过身，往回爬去，想拖回那半截身子。 他从未像现在这样努力，他直直地伸出手去，要去抓他那下半身的一只脚。 终于，他抓到了那只脚，但却再无力将这下半身拖回身边。 他的目光越过眼前自己的残肢，看见了大狼。那最后的一次发力，使得它那伤口裂得骇人的大，血如地底的原油一样高高喷出。它的胸口起伏得剧烈，像是做着最后的挣扎。 桑丘的感知如大狼的血一样，从身体里飞速地消失。 救救它，谁能救救它，它就要死了，快来一个有善心的人呀。桑丘含糊地想着。 眼前渐渐地黑了，只有那断了的麦子被风吹动，黑压压地从桑丘的眼前打过。 还有十天，就可以收割了。难道人们忍心任其烂在在地里？想到这，桑丘痛苦得浑身颤抖。 这由麦子引起的最后的伤感终于要了他的命。
