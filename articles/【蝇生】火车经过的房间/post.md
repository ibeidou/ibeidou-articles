# 火车经过的房间

大学时期我不喜欢宿舍拥挤的环境，曾短暂地租住在一处紧挨着火车铁轨的小房子里。每天三十多趟火车从我的窗口呼啸而过，带来一阵凉凉的铁味儿和一次微型地震，桌上的稿纸一下子飞得到处都是。好在这里租金便宜，也不用与别人分享卧室和洗手间。我逐渐习惯了火车日夜经过的感觉，甚至有点喜欢上了它。

[![huochjgdfj1](f9d8b51e-7cf3-45f1-808d-f9239d8f0f0a.jpg)](b830f7b1-025f-4a3d-b3d6-297c368a0c42.jpg)

那时候我一边上学，一边给一家卖通讯设备的公司做语音客服，负责晚上10点到凌晨2点的顾客售后服务。这家公司主要是卖私人卫星接收器的，俗称“锅”，用来截获全球的卫星电视信号。我从来搞不懂这些玩意儿，不过没关系，所有的问题和答案都已经完完整整地印在了十几页A4打印纸上，连语气词都已经写好。我所需要做的就是从这十几页纸上找出问题，然后一字一句地念出来：“您好先生，请问有什么可以帮您？”“好的先生，您的问题很常见，有可能是机芯过热导致的……”

说实话，半夜打电话来的人真的很少，我经常一边看着美剧，一边忘记了时间。偶尔有一两个电话打进来，我也喜欢敷衍了事，客气地记下顾客的地址和电话，推给上门服务的技术工程师。当然会有骚扰电话，这是每一个深夜上班的客服都会面临的处境，有些甚至很下流。不过我看得挺开，毕竟在这寂寞的城市里，我还有电脑作伴，而他们可能只有恰巧坏了的“花花公子”电视台。

[![huochejgdfj2](4c709fa8-2059-4c9c-982e-da4744035d81.jpg)](c35b56cc-9bbd-4a58-8c8f-ba483197bcd3.jpg)

我是在一个周五的夜晚接到周先生电话的，他的声音浑厚，听起来有四五十岁了，普通话不大标准，有一股港台腔。他说他的锅坏了，无法接到信号，我查阅了回复指南的第十页，一步一步指导他重新连接。没用，还是无法正常运作，大概是真的坏了。我记下他的电话和地址，告诉他明天会有技术人员上门维修。他显得很焦虑，问能不能现在过来。

这也不是第一次出现这种情况了，有些顾客需要收看特定的外国电视节目才能睡觉，我还遇到过某某功的弟子声称不按时看节目就会下地狱的恐吓。不过没办法，我只能耐心地安慰他，并承诺明天一早就会有工作人员与他联系。周先生轻轻叹了口气，突然问我叫什么名字，我愣了一下，告诉他我叫小猫。这当然是假的，不过他没多问。他说小猫，真是个好名字，我们台湾农村就喜欢给女孩子起名阿猫阿狗的，好养活。

他接着说，小猫你有多大，听着也有二十多岁了吧，你知道周杰伦吗？我说我知道，大家都知道。周先生说，我就知道你们年轻人喜欢他，我告诉你啊，周杰伦是我儿子。我来了精神，赶紧竖起耳朵听下去。他说，我和他妈妈离婚早，这孩子命苦啊，以前我糊涂，做了对不起他们母子的事，到现在他都不原谅我。小猫，你有爸妈吗？我不是那个意思，我是想问，你和爸妈住在一起吗？应该不是了，我要是你爸爸，肯定不让你这么晚睡觉，对身体不好。不要嫌父母唠叨，他们也是为了你好，明白吗？要好好珍惜，多和父母说说话，不要不理他们。爸妈年纪大了，跟不上潮流了，但是他们还是喜欢听你讲话的，你明白吗？

这番话我听得一头雾水，不过好像很感人的样子，让我想起我自己的爸，这个点儿他说不定正在外面打牌。周先生接着说，小猫啊，你知道我为什么要装这个锅么，就是为了看我儿子。我离开台湾十多年了，年轻时为了一个女人，抛弃所有来到大陆，结果现在什么都没有，连儿子都不认我了。我不怪他，是我犯了错，我不该打他妈妈，也不该抛弃他们。我现在只能每天在电视上看一看他，就好像他真的在我身边一样，这种做父亲的心情，你明白么？

[![huochejgdfj3](36e92c40-9f87-4893-86e9-1dc637705975.jpg)](b9162a9b-db2a-4d9e-bc5c-a42211346eac.jpg)

我用力地点点头，眼泪在眼眶里打转，我也有不幸的童年，爸妈年轻时关系不好，经常吵架，这些事像是一道伤疤，在心情的阴雨天气里隐隐作痛。周先生顿了一顿，好像很伤感的样子，他说小猫啊，我本不想麻烦你，可是我儿子的演唱会，一点钟就要开始了，我的锅坏了，能不能麻烦你来我家看一下啊，我的地址是厦门市思明区⋯⋯

啪的一声，我挂掉了电话。一辆绿皮火车轰隆隆地从窗口经过，新鲜的风灌进房间，我一个人，在这过分安静的午夜里，像个神经病似的放声大笑了起来。

（采编：陈静；责编：王卜玄）

[【蝇生】商场的地下王国](/archives/40686)——在位于商城负二楼的书店，只要畅销书的读者来过，一线明星来过，买书装饰海景房的贵妇来过，求婚的人来过，读者来了又去，谁见证了书店的繁盛与落寞呢？

[【蝇生】杀人公司](/archives/40630)——“农人”是行业里对收尸人的黑话，到指定的地点收尸叫“收菜”，杀手叫“厨子”。大部分厨子都是女人，因为女人好伪装，不容易被发现。另外，女厨子心思细腻，不爱留马脚。事实上，农人平时也收尸，在火葬场干活，厨子平时可能是任何人。

[【蝇生】天仙配](/archives/40761)——除了歌颂他们是城市建设者，听闻他们讨不到工资自杀，你了解过农民工另一面吗？他们偷女人内裤，暴打工头到屎尿横流，除了猥琐和暴力，他们的人生，还有另一种可能吗？

[【蝇生】保安](/archives/25973)——我时常需要在这傻逼世界中探出头换换气，却不敢将鳃长久暴露于寒冷稀薄的空气中。这时我总会怀念起那个半睡半醒慢悠悠的人。早忘了自己啥时候开始认定这个世界很傻逼，但好像已经不重要了。
