# 泡澡堂子

芜湖是个不南不北的地方。说北，却已经是江南；说南，冬天也冷到零下。小时候都只有窗式空调，只能在卧室发挥功效，一到入冬的几个月，在家里洗澡总是嫌冷，稍不注意，就难免伤风感冒，所以芜湖人也是泡澡堂子的。



泡澡堂子既然是“泡”，要的就是大池子。澡堂子墙多半刷白，水池使用蓝矾杀菌，十几号人如下饺子一般泡在池子里，氤氲缭绕，雾气腾腾，热气把你蒸得大汗淋淋，有一种酣畅淋漓的感觉。时而有人加水，有人递热手把子，遇上三五熟人，泡一壶茶，一碟花生，扯些闲篇，张家长李家短地乱吹一通，高兴起来哼几句黄梅戏，其乐融融，好不自在。



后来我来到了东北，学校的浴室是没有大池子的，总感觉这是冲凉，少了点洗澡的情调。待到哥们引我去南岭小街的一家热水浴，我这才开始在东北泡澡。这家浴池不甚大，但也足够容纳二十几号人同时洗浴。其特点在于除去北方共有的蒸汽房外，还有四个大水池，其中一个是冷水池，三个是不同温度的热水池，任君选择。上完班一身疲惫的老少爷们，在热水池里泡上五到十分钟，一天的乏都在烟熏缭绕中散去。要是对自己的心肺功能有自信，可以采用冷热水池来回交替的“冰火两重天”，在血管的扩张与收缩的过程中正确的呼吸吐纳，可以导气冲虚，强身健体。长期冷热交替泡澡的人，往往抵抗力强，精力充沛，也更为持久。这家浴室大众消费，三教九流均可入浴，而且干净整洁，就算你是达官贵人，来洗也不算跌份，而且温度够热，汗出的够多，比某些高档洗浴会所还洗的过瘾，所以每天顾客盈门，周末更是人满为患。



除去大水池子外，这家浴室也和其他的一样，搓澡、修脚、拔罐一应俱全，这也是澡堂子文化的重要成分。常去的熟客，和师傅们早已混熟，在搓澡捏脚的时候吹几句、扯几句，谈谈吃、讲讲情。澡堂子的师傅见过各式各样各行各业的人，加之东北人本身就开朗好说，他们的口中总是有说不完的故事。在象牙塔里关久了，到澡堂子里接接地气，听听父老乡亲讲带点颜色的笑话，也是种不错的消遣。

管你是什么身份，贩夫走卒也好，大学教授也罢，在澡堂子脱了衣服，赤条条白花花，大家都只是哥们。衣服就是戏服，当人穿上衣服，就有了分别，阶级，地位与行业，于是人根据自己的身份扮演自己的角色，脸上言笑晏晏，内里甘苦自知。脱去衣服光溜溜，也就等于脱去了面具，什么领导知识分子，都不过是一堆赘肉，还没有劳动群众来的健美。所以在水池子里，大家都没有什么界限，话说得总是投机，因为没有行头，没有面具，谁也不认识谁，没有必要虚情假意，一起扯淡，一起放肆的大笑，生活的压力与艰辛都随着热气与汗珠流走，这一刻，才是岁月静好，现世安稳呵。天南地北地海吹，有一搭没一搭地聊着，说什么的都有，但不外乎政治与感情。也难怪，政治与感情的相同点都是表面上冠冕堂皇道貌岸然，喜欢用神圣的外衣自我加冕，暗里却冷暖自知。不过是绣花枕头的装点着甘苦与权术的计较。当算计绑架了社会，这便是政治；当欲望填补了生活，这便是感情。人一旦与面具签订契约，便会反过来被面具控制，不能自拔。黑色幽默与黄色笑话，便是以逗乐的方式褪去外衣，不过是说说实话而已。却总能搏得被面具挟持的人们会心一笑。笑够了，也洗好了，从水池上来，汗珠还附着在肌肤上，找师傅拔个火罐，小眯半个小时，真个是云我共悠悠，舒服赛神仙。然后穿上衣服，汇入人海，继续扮演自己的角色，唱好自己的戏码。人与人永远无法真正平等，因为人穿衣服，这便是文明，不过是一场大规模全人类的角色扮演。



一起去洗澡的哥们，往往关系都不一般。看男人之间的感情如何，就看他找谁一起吃烧酒，找谁一起泡池子。应酬场合华灯炫目，觥筹交错，说的都是些场面话，一桌十几号人，各打自己的小算盘，还要拼酒灌酒，让人无法生出好感。而叫上两三知己，也不必去什么大饭店，就找那街边排挡，来上几十串烤肉，喝点烧酒，说说体己话，肉大口的吃，酒喝多少看兴致，约的都是兄弟。而能一起去洗澡，那更是“赤诚”相待，对你毫无保留；领导如果能叫上下属一起泡澡，那必定是恩宠甚厚高看一眼；前辈能叫上后辈一起泡澡，那必定是青眼有加推心置腹，关系密的嘞！



泡澡堂子是闲情逸致，急是急不得的，只来个三十分钟一个小时，匆匆而来匆匆而去，那只是冲凉，给自己拾掇拾掇，为了更好的演戏。东北人好享受生活，喜欢泡澡堂子，只是泡澡的大多都是中年以上的年纪，年轻人却很少。只是现代人日夜匆匆忙忙，哪有这样的闲工夫，只是日夜忙来忙去如没头苍蝇一般，到底图个啥，想必内心也是一团浆糊。不如来泡泡澡，在烟雾缭绕中理清思绪，说不定会更好。



泡澡，在大水池的热气腾腾里，老少爷们的大言凿凿中，便是人间的情致。
