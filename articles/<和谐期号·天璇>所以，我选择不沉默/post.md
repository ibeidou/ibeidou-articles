# 所以，我选择不沉默

## 文 / 吉米糕（中国政法大学）

 近来偶尔听到这样的话，你怎么就变这样了？ 而当这样的问候来自法大校友时，我在笑着的同时心里会小小的酸一下。 我的母校，中国政法大学，我从不低估它在中国法律人职业共同体形成中所起到的重要作用，更关键的是，它以自己和诸多前人先辈的风骨告诉了它的后继者，在中国这片神奇瑰丽的土地上，无论理想走向现实的路程有多艰难，总有什么是值得我们坚持并为之去付出甚至牺牲的。 我只是其中的一个普通人。经历过懵懂，无知，彷徨，挣扎，抉择，这其中的很多，在之前的一篇文章里，我曾回忆过。( [《我在法大的信仰之路》](http://blog.renren.com/blog/227775927/389916337) ) 有些事情却变化了。多年前为保护学生下跪的老师，如今作为中国法律界信仰神坛上孤独的老者被高高供起，更多的老师经历了多年前的挫败后转入拜金主义和虚无主义的大军。摇旗呐喊的学生，坚持的，没落的，背叛的，威权体制下不能祈求人性的光芒可以时时绽放，而当那反面和愚民教育相结合时，一代又一代，面对皇朝卑微的臣民和面对异邦愤怒的拳头，成了盛世中华的最多数写照。 教育产业化，贫富差异在教育领域只是更显出其凶残和不公。教材弱智化，历史被篡改或践踏，孩子自小价值观的扭曲，为多年后民意被操控埋下伏笔。一切的根源却不只是教育。民工子弟学校的学生，代课三十年手攥几百元钱被清退的乡村教师，地震中永远被深埋的弱小生命，和为了他们奔走却锒铛入狱或远走异国他乡的热忱的爱着这个国家的人们，中国遗忘的和忽略的和亏欠的，正在以越来越多的不幸作为注脚。 2009年，一个悲剧继续上演的年份。自焚者，上访者，律师，公共知识分子，普通公民，无论你是否有意识的参与着国家进步或倒退的历程，在这个体制之下，无人可以逃脱。沉默的大多数总是觉得自己可以避免那些灾难，却不知道今日你所容忍漠视的一切，明日就可能真实的在你身上上演。 和母亲聊天，我非常理解她的担忧和不满。 有一句话，我昨日说，今日说，明日还会说的是： 我今天做这些，就是为了将来某一天，当我面对我的女儿的奇怪言论时，不会有你今日的恐惧。 就在这之后几天，我的校内账号被封，因为我连续四次发了同样一篇日志。在那篇日志里有一位老师姓刘，有一群人叫做司法机关，有一个数字叫做11。 一个因言获罪仍然纵横无阻的国度，一个心性被犬儒和亢奋和仇恨主导的民族，很容易让人绝望，决意放弃。 没有人天生具有抵制恐惧的强大力量。我一样会恐惧，不只因为我是女儿家，还因为我珍视此世上的许多事情，我一样期待安稳的生活，我爱我的父母，我的爱人，我的朋友，我爱这世上的许多美好，我热爱读书，写作，唱歌，旅游，美食。 在得知账号被封后，我经历过很多种情绪。失落，无助，激昂，愤恨。自诩为价值观足够坚强的自己，在庞大的威权阴影下，仍然会短暂的手足无措，甚至想过，为何要我来承受。 我也问过自己，你可以沉默吗？ 沉默可以带来很多好处。对苦难无视，便可以使自己时常保持快乐的情绪，有助于身体健康。对体制顺从，便可以在既得利益中好好的分一杯羹，孝敬父母。作为知识人并擅长写作的我甚至可以比较轻松的拿它去换取财富，无论产生的是垃圾抑或毒药。沉默的鸵鸟这世上已经太多太多，难道还少我一只不成？ 我可以选择遗忘，安稳的生活。和爱人一起为了生计奔波，老老实实地赚钱，买房，买车，那剥夺了一代中国年轻人理想的房价，已经足以让我忘记其他。我可以在老去的过程中尽情享乐，享受这个国家现代化的成果，只要压榨的不是我，我管他什么底层人民的受苦。我可以尽我所能让我的孩子拥有在北京接受教育不被歧视的可能，我可以让我的孩子接受尽可能的好的教育，我甚至可以为他请一个私塾先生，免受学校教育的不良摧残。 只是，多少年后。 我的孩子会问我。 --妈妈，为什么房价还是这么高？为什么我们只能上大中华局域网？为什么我们的国家和大多数国家不一样却还是那样理直气壮？为什么我们经济强盛却还是赢不得尊重？为什么我要怕警察怕城管怕国宝？为什么我们还是被代表却没有自己说话的权利？ 我该怎样回答他（她）呢？ --因为几十年前，我们错失了政治改革的良机，从此在一个失去民主制衡的经济改革中大多数民众成了既得利益集团的挤奶工。 --那为什么不能继续下去呢？ --因为几十年前，妈妈和爸爸因为恐惧而失声。而像我们这样的人，是中国的大多数。没有人说话，就觉得一切都还不错了。 --那为什么你们恐惧呢？ --因为几十年前，你的父辈太怂。 我无法面对这样的对话，所以，我选择不沉默。 在历史久远的黑暗里，就是这样一束束罅隙里透露的光，最终托起明日的太阳。 写完这些，我发现自己仍是个坚定的理想主义者，我很欣慰。
