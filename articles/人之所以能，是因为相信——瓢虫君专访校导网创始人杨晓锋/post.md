# 人之所以能，是因为相信——瓢虫君专访校导网创始人杨晓锋

杨晓锋，武汉理工大学工商管理系09级学生，同时也是校导网创始人兼首席执行官、武汉梦不落文化传播有限公司董事长，曾任武汉华夏飞马信息技术有限公司董事长兼首席执行官。

他是如何做到今天这一步的？他经历了哪些抉择，走过哪些彷徨与迷茫？今天的他又怎样定义自己的成功与未来？

_瓢虫君专访大学生媒体创业人杨晓锋__—__带你走近他的年轻与挣扎，收获与感悟。_

[![yangxiaofeng](2aa6ab7f-6148-4fcc-aa2f-39da6c0f7677.png)](d1938554-f894-44e3-a670-570b827dbb45.png)

**“在创业之前该干的事太多了”**

08年5月汶川地震，杨晓锋只能捐出800元，他突然意识到“必须要让自己强大起来”。几乎同一时间杨晓锋成立了武汉大学生飞马创业团队，不出半年，这个团队净赚10余万元。

最初的成功只是因为热忱的理想与些许运气吗？

2006年杨晓锋高考失利。于父亲而言，能上一本就是好事，心有不甘的杨晓锋却决定复读。父亲坚决不肯出生活费（一本线上学生复读不用交学费），暑假打工挣来的钱终究是杯水车薪，迫于无奈，杨晓锋找到校董，希望学校可以承担自己的生活费。校董直接拒绝了他。杨晓锋开始分析学校承担生活费让他复读可以带来的好处，他利用提高升学率为由说服校董，并承诺担当班长、充当班主任的角色，主动讲英语课、举办各种团结班级的活动。杨晓锋最终实现了自己承诺，同时也使班级中的更多同学考上了大学。

这场小试牛刀，为杨晓锋举办“志愿服务西部三项下乡系列活动”奠定了心理基础。“做这个活动可以说是想锻炼自己，就如同我当初复读，不过是向父亲证明我的实力很优秀，同时我自己很希望做一些志愿工作。”

做好活动需要宣传，杨晓锋硬是说服了电视台台长，借助当地的电视台，为自己播30秒的广告。之后，杨晓锋又找台长商量要将此事播进新闻。“当时我每天都在家里看那些有关说服力的书，去学习当中的技巧。”

这种说服技巧还被用在与城管的交锋上。为了那次志愿活动，杨晓锋拉到了不少外联，甚至将宣传横幅挂在县政府门口，结果被城管没收了。于是他晚上看书学技巧，白天就去找城管“做实验”，一种方法不行就换另一种，终于成功说服城管局长，要回了横幅。之后为了让家长放心，杨晓锋自学写合同，让每个学生带着签好的合同回家。

杨晓锋还向曾经与自己一起读高中的大学的同学做宣传，让那些愿意做义务家教的大学生来这里。因为甘肃气候条件不太好，不少大学生水土不服，杨晓锋又做起了后勤保障工作。他还专门学习备课、安排课时的知识，确保学生们可以学到真东西。为了让大学生们的暑期实践表能盖章，杨晓锋又多次去教育局和县政府协商，在数次碰壁之后杨晓锋转战团委，成功拿到章子。杨晓锋顺便也给自己盖了个章，就这样他进武汉理工大学后顺利进入南湖校区学生会外联部，积攒下第一份人脉关系，而此后杨晓锋带领别人兼职的经验也为他建立飞马团队打下基础。

**“梦不落 为梦想而战”**

到大二上，飞马团队的发展已远远超过了杨晓锋的想象，为了更好地发展自己的事业，杨晓锋决定休学创业。他说服了学校和父母同意休学，实际上却背负了满满的压力和愧疚感。“感觉自己千不该万不该这样折腾，像是犯了一个不可饶恕的错误。”

资金不足、管理力不够、心态不成熟，这是不少大学生创业时不可避免的问题，杨晓锋虽有这个考量，但还是决定“先打枪后瞄准”。考虑到传统领域需要一定资源、人脉积累，杨晓锋选择在互联网一展身手。年轻气盛的杨晓锋急于干出点名堂证明自己，没有一个完全清晰的目标，在茫茫商海中寻求自己的一席之地，他历经了一次惨败。

这段惨痛的经历使杨晓锋看清真正的朋友，直面残酷的社会，“不是沉默就是爆发，不是灭亡就是重生。我正接受着前所未有的考验和挑战，感觉整个人在加速的衰老、日益的憔悴，有时候甚至是有瘫痪的感觉，每天总觉得被什么东西压得喘不过气来。” 他开始反复思考着人生的意义和价值，度量坚守和承诺的代价。在采访时，杨晓锋尽量不说这段历史，“我不希望父母为我担忧。”

此时朋友李炎的出现帮助杨晓锋渡过难关，经过业务转型，成功救活公司。杨晓锋在公司业绩蒸蒸日上时选择回校复学，因觉自己不合适材科专业，转入工商管理。在读书阶段他又琢磨出新点子，于2011年6月建成校导网的初版页面。

“核心竞争力不是面面俱到，不是超级巨无霸，而是一件事做到无可挑剔无人能敌，哪怕是打扫卫生或者洗马桶。”2012年5月，考虑到当时的校导网包含太多东西，不利于初期创业公司运营，杨晓锋决定将其定位重新聚焦于学生社团服务。“这是个商业决定，但同时只有做到专注、聚焦，让别人没法抄袭，才会有核心竞争力。”

“选择大学生作为我的消费群体，是因为我本身也是大学生，我了解他们的需求。”杨晓锋坚信大学生是民族的未来，为大学生服务是一件光荣的事情，“希望可以引导他们有正确的价值观。”大学生就业是一大难点，杨晓锋将就业作为一大重要版块在校导网上运行，一方面帮助大学生和社团走出校园，参与实习就业和社团文化演出；另一方面帮助企业走进校园，实现人才招募和品牌宣传，最终实现校园和企业的合作交流。

“如果能够选择，可能我会等自己更成熟一点再创业，当初还是有些急，自己不愿意放弃正在日益壮大的团队。不过我并不后悔这条路，因为我宁愿在创业中奋战，也不愿做一个普通的学生。这是我的梦想，所以我坚持。”



**大学生群体 ****印象 ****答问**

**Q1 ****：**“大学里从大一到大四，我发现有几样东西很容易成为过去时——激情、梦想、信念等，这也使得我跟大家分开来，因为我不想这些东西成为过去时，希望他们永远是进行时。”这句话可以算是您对大学生的最本质的忠告吗？ **A ****：**是的，虽然这一点对不少学生来说很难坚持。但是只要肯坚持，就会有很大的收获。 **Q2：**大学生需要培养的到底是什么？ **A ****：**素质。这种素质不是一般意义上的那种，而是认识世界、改变世界的能力。人活着是为了改变世界，这才是大学生需要培养的意识。有了这个意识，他们才能更清晰地面对未来。 **Q3：**休学期满，您为什么选择回来复学？这是不是代表着您对学历的一种看法？ **A****： **选择复学一是当时正好休学时间满了，也该回来复学；二是我父亲认为培养一个大学生是很光荣的事情，我不能让他失望；三就是学校带来的不仅是知识，还有人脉和平台。至于我转专业，是因为我觉得我更适合读工商管理，而它的确也很有用。至于学历，其实是一张门票，而你个人有没有价值，不是这张门票所能完全决定的。 **Q4** ：您怎么看待大学生目前的学习状态？ **A ****：** 我觉得现在大学生学习状态普遍不好，一方面是因为现在的教育与社会发展有脱节，但同时学生们身在信息社会，应该自己学会掌握信息，决定自己的发展。所以说大部分因素还是现在不少大学生没有钻研精神。他们的实践能力太弱，自然会越学越觉得没意思。 **Q5****：** 靠着梦想走到今天的您，在实现梦想的过程中有没有遇到对现实妥协的事情？ **A ****：**两点之间最短的不一定是直线，解决问题也不止一种方法。当能力有限的时候，难免会有所妥协，但是这些妥协都是在不违背原则的情况下实行的。这种妥协并不是软弱的代名词，而是为了强大，为了让以后拥有话语权，为了让以后不再妥协而做的。 **Q6****：**除去创业者的身份，您最单纯的大学生生活是怎样的呢？ **A ****：**只能说自己不算是一个好学生，为了公司难免要牺牲自己的学习时间。不过和同学的关系都很好，他们都叫我锋哥，有些事情也会问我。现在正在准备毕业论文，跟最普通的学生一样，说说网络流行语，上上课，做做作业。 **Q7****：** 官博中您写了一篇《写给二十几岁的女孩子》，非常适合女生去看一看， （[http://home.xiaodao360.com/blog-14-18.html](http://home.xiaodao360.com/blog-14-18.html)），对于这个年纪的男生，您又有什么建议呢？ **A ****：** 那篇文章其实是写给妹妹看的。如果说男生的话，应该有高瞻远瞩的思想。不能够局限于眼前，要能承担更大的责任，比起女生更加有使命感。我觉得在大学里有比游戏更有趣的游戏，即创业、志愿服务、科研。 **Q8****：** 您当初休学时，父亲以为自己培育出的大学生“夭折”了，南方周末在3月1日，发布一篇名为《你必须得上大学，但是……》的文章（[http://www.infzm.com/content/88435](http://www.infzm.com/content/88435)）反观现在一些三本或者更差的大学，只是为了捞钱，一些贫困大学生在那里读书，但仍然逃不过找不到好工作的命运。这类事您是怎么看待的？ **A ****：**这是一个社会问题，只能说希望国家相关部门能规范学校审核制度，教育大纲和政策。作为青年的话，如果还没有上，就没有必要上了。上了这种大学也不要灰心，只要肯努力，肯利用资源，还是会有好结果的。 **Q****9****：**“如果一个人努力地工作，努力地学习，却还是不能改变命运，依然贫穷，”那么这样的群体，他们该做些什么呢？ **A****： **这只能说他们的方向错了，这时候最重要的是停下来，认真思考自己真正的方向，并为之努力。 **Q****10**：校导网中有一个版块是校园资讯，这原是传统校园媒体应该提供的服务。而今各大学校的校园媒体都呈现低迷的状态，您对校园媒体有什么看法和建议，它们的发展前景何在，尤其是校园文化这一块？ **A ****：**现在的校园新闻质量普遍比较偏低，也没有什么新闻点，或者说新闻点不够新。而且各个校园媒体之间也是层次不齐，感觉彼此间缺少交流。至于校园文化，感觉缺乏创新，一直都是我行我素的状态。 **Q11****： **北斗网是一个全志愿性的大学生思想交流平台，意在打造大学生人文生活网络氛围，引领大学生自我教育、自我启蒙。面对北斗网的全志愿性形式，它似乎比校导网更加理想主义，商业化与志愿性的选择，您是怎么看的呢？ **A****： **这个是商业与公益的关系。商业是解决生存问题，公益是在生存问题解决以后更高的精神追求。吃饭是为了活着，但活着不仅仅是为了吃饱。有了能力，就可以提升自己在社会的地位，也就有说服力，能够好地为社会服务。    **记者：叶馨文** **责编：郑萃颖** 北斗人物投稿邮箱：beidouqxrw@163.com
