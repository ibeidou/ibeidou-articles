# 脑电波

他醒来的时候发现自己在别人手机里。下意识想要摸摸头的时候发现自己没有手，想要动弹的时候发现自己根本就没有身体，想要恐慌的时候却发现，根本没有“自己”这个实体。

还没等他弄明白，就听到一个中年男子疲惫的低声，那声音让他想起小时候音乐教室里闷声闷气的脚踏风琴。这台风琴死气沉沉地踩踏出一串嘶哑的前奏：手术失败了，曾伟死了。

好似又遭受了一次除颤器的高压电击，他的记忆在众人齐迸的哀嚎中猛然苏醒。没错儿，自己叫曾伟，是个人，死的那个，好像就是他。

现在这情况让他不由得想起只会唾液四溢侃大山的音乐老师，如说日本人发明了一种铅笔头大的鸡蛋，往水里一扔就爆炸成正常大小；前苏联人发明了一种可融进身体的枪，拍打后脑勺就会从脚后跟出来；还说美国人发现，人死后脑电波不会消失，会在空气里停留一段时间。总之各种荒唐事都被他说的有鼻子有眼的。

那他现在就是一串脑电波。

还没等曾伟想明白，就接收到一串电波。那电波变换着如USB接口一样接上并告诉他，他现在确实是一串脑电波，在给他做手术的医生的手机里，还有，这支手机里不止他这一串。

那串电波叫崔宁，已经在手机里呆了六十四个小时。做完手术醒来后，就发现自己已经变成一串脑电波，还被吸收进手机里。

曾伟想起，手术前几天坐电梯时，一具蓝色塑料布裹着的尸体被推进来。平时低眉顺眼的保洁那次格外神气：“全部给我出来！”从早到晚都关不掉大嗓门的病人家属，那一次都默默地让开。

“可能是我。”崔宁突然传给曾伟这样的颤动。曾伟惊讶地发现，他们可以读到彼此的思想。

崔宁告诉曾伟：“我们不知道什么时候就会消失，你有什么没交代的可以通过手机QQ告诉家人，前几天我就把自己的银行卡密码告诉媳妇儿了。”

曾伟陷入沉思。家里的大小事都是妻子在管，银行卡、存折这类事，他压根不操心。除了每天按时上下班，每个月按时给工资，每年给妻子一点丈夫该给的红利，其他的，他对这个家，好像没什么特别贡献。

贤惠的妻子却总是告诉儿子，是他撑起了这个家。与自嘲二等残废的他不同，儿子除了浓眉大眼，还人高马大。就连青春躁动期，也乖巧聪明如故，没叫他费过半点心。

因此同学聚会的时候，他总会用各种借口带上儿子。在老同学的艳羡声中，他的脸上总是傻呵呵甜蜜蜜地，浮现出一片红艳艳的幸福。

结果出来后，妻子呆坐在他们那台本命年色的汽车上，悄无声息地湿了眼眶。车子有些年代了，车门车窗都关不严，风从四面八方的小缝隙里细溜溜的灌进来，冻得他直哆嗦。

手术是妻子安排的。医生是妻子曾经的同学，曾经的全校第一。妻子说，她忍受不了半分钟的耽误。

在崔宁的指导下，曾伟用医生的手机上了QQ，没想到刚打开，妻子的头像就在晃动着一堆缠绵的情话。

正纳闷时，妻子含情脉脉地问了一句：“今晚上你那还是我那？”

没等他反应过来，妻子又发来一句：“还是来我这吧，儿子和同学出去玩了。对了，咱们儿子这次考的好，让他上咱们的母校吧。”

崔宁突然感受到一阵强烈的波动，开始分崩离析。即使脑电波连着脑电波，崔宁也没读懂他俩消逝前，曾伟那激烈的颤动。

完成一桩手术后，医生在办公室里迫不及待的掏出自己的手机，见到QQ上的话，不由自主的，两道弧线如漫不经心的手术刀般划弯了他的嘴角。他飞快地发出一串甜蜜，笃定地等待着，嘀嘀嘀的声音送来一条条让人心颤的电波。
