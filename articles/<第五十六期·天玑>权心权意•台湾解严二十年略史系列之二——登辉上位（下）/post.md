# 权心权意·台湾解严二十年略史

# ——系列之三·登辉上位（下）

## 文/周远游（天津大学）

**第三折 登辉上位（下**） 说起来李登辉，无疑这是一个争议人物，相信大多数人，本能的把他同死硬台独分子，汉奸等等不好的称谓联系在一起，一说起此人，不少人就如同打了鸡血一般，浑身发烫。 我无意于给阿辉翻案，因为每个人基于其背景，教育和生活经历，必然对历史对历史人物有自己的理解，人的立场不是靠文字能动摇的，如果动摇了，那么只能说这是风派人物，没有任何独立思考能力可言。在我眼中，一个固执的死硬派，比起一个灵活的骑墙派要可爱得多。 因此，我所说的，都是我能看见的材料，文献，以及根据我个人的人生观和方法论得出的结论，大家可参考，仅此而已。 李登辉，也许真的不爱广义的大中国，对狭义的中国人没有认同，但是任何人也无法否认，李登辉比起他之前的任何在台的领袖人物都要爱台湾，爱着所谓正港台湾人。 李登辉也许真有浓厚的日本情结，但是这个日本情结，是不是和我们理解的媚日是一码事？李登辉的书写的很清楚，如果还有疑问，建议看蔡康永的《真情指数》对阿辉的专访。 还是要多嘴一句，是马关条约把台湾割让给鬼子，是祖国母亲对不起台湾在先，在日军登陆的前十年，台湾人宁死不屈，掀起了一浪高过一浪的抵抗运动，祖国母亲管了么？支援了么？面对穷凶极恶的日军屠杀自己的同胞坑气了么？ 台湾的统派网友，比如yst等人，说所谓正港台湾人，也就是占台湾总人口75%的福佬人，都是畏威而不怀德，只会以多欺少，恃强凌弱，阿谀美日，看不起大陆 同胞，将人类劣根性体现的淋漓尽致，我承认，现阶段，这说的是事实，这群人就是这德行，那是因为，有勇气，有格局，有心胸的台籍精英，在那个祖国母亲抛弃 了的年代里，基本被日本人残杀殆尽！算起来，祖国母亲是不是有责任？ 这就好比，妈妈因为艰困，将孩子送给了仇人，孩子虽然年幼却誓死不从， 不要认贼作父，结果被贼人日夜毒打，无论孩子怎么哭嚎，妈妈不但不理，还装作不认识，终于，孩子被打怕了，屈服了。而多年后，妈妈靠着外人的帮忙，要回了 孩子，可是却不把孩子当亲生的，照样虐待，而没几年，家里还换了女主人，侧室成为正房，大妇躲到外宅，除了细软和怨气，什么也没带来。在这种情况下，你让这个孩子忘掉过去的不快，不准抱怨，不准回忆历史，还高高兴兴的去和这个家过日子，嗯，我只能说，能做得到是情分，做不到也是正常。 啊，貌似又跑了，我们继续吧。 本来，李登辉接总统，没有任何可说的，宪法规定嘛，那个任期还有两年，所以不必改选。 但是，如果李登辉拿不到党主席，在威权时代没有党机器和党资源的话，那么可以断定，90年的改选是一定会被换下来。 不要忘了，彼时选总统的，不是选民，是国民大会，所谓万年国大，这还是48年时，中国第一次真正意义的普选选出来的国民代表，这些人平均年龄已经70多了， 多年没有改选，目的自然是为了维持roc的法统。在这个外省人，保守势力，蒋家遗臣占多数的国会下，如果李登辉没有相应配套资源，仅仅空有总统头衔，那是没有胜算的。 因此，李登辉别无选择，必须一举拿下国民党主席。 在蒋经国去世的当晚，国民党立即举行了紧急中常会，行政院长俞国华主持。在会上，他宣读了蒋经国的遗嘱，同时议定三件事： 即全体国民党“恪遵蒋经国遗嘱”； 李登辉依宪法继任总统； 立刻展开治丧事宜。 当时，俞国华等元老派曾打算党主席一职拖到7月7日召开十三大时解决，在这以前由国民党中常委轮流主持中常会。据信，这个提议得到了宋美龄的支持。 可以说，这对于李登辉，是极为不利的，一旦对手按部就班完成部署，身边几乎无人的李登辉将没有翻盘的机会。 而无疑，李登辉最大的对手，就是已经90高龄的宋美龄，宋美龄在蒋介石死后一直居住在美国纽约长岛，一住就是11年。她是在蒋经国病情加重，蒋纬国出任国安会秘书长之际却回到台湾。而蒋经国也默许了宋美龄重新进入圈子，因此，宋美龄是所谓官邸派的龙头，此时她的官衔是中央评议会主席团主席，众人称夫人而不敢名，在台湾政坛的影响力依然非比寻常。在她周围有一批国民党元老如黄少谷、倪文亚、谷正纲、沈昌焕、郝柏村、宋心濂、秦孝仪等人，基本上，外省人都寄希望于宋美龄在此时能站出来，主持大局。 宋美龄先是属意李焕，但是李焕此人曾经受过重挫，复出后声势大不如前，而且最要紧的，李焕属于改革派，在政见上和李登辉一致，他也认为李登辉接班比较合适，并无争雄之心。 接着，宋美龄还想到了俞国华，俞国华倒是愿意接班，但此人守成有余，进取不足，这也是李登辉最后能战而胜之的根本原因。 当然，事情关键，还要看阿辉自己。 阿辉的第一步，就是在接任总统后，第一时间前往宋美龄处输诚，目的自然不言而喻，而我们知道，当年经国先生在时，李登辉坐沙发从来都是只坐三分之一屁股，谦恭的很，而及时拜访这一举动也得到了宋美龄的好感，无形中争取到了相当的时间，也让不少党国大佬们对阿辉观感大为改善。 接着，李登辉马不停蹄，及时拜访严家淦，张群、陈立夫、谷正纲等国民党元老，表示自己多年追随蒋经国，今后要遵循蒋氏父子遗愿。同时，他此时一有空就往灵堂跑，在蒋经国遗像前沉思默哀。这些举动有多少诚意姑且不论，但是确实在当时，受到了良好的效果。 最关键的，李登辉选择了拥抱少壮派，具体时间如今已不可靠，但是大概在1月17左右，李登辉和一个重要人物见面了，这个人就是时任国民党副秘书长的宋楚瑜。 他们谈论了什么，如今已经不知道，他们是谁找的谁，如今也不知道，因为李宋二人如今各执一词，就像罗生门一样，但是无疑，这是李登辉最关键的一步。 稍微熟悉台湾的人都知道，宋楚瑜之所以日后如日中天，就是因为在李登辉上位一事上起到了临门一脚的作用，但是大家也许不知道，真正点燃这把火的人，恰恰也是另外一个反李反台独最激烈的人。 这个人，我们先前提到过，就是所谓政治金童，反台独的急先锋，日后的新党总召，如今的媒体大佬，赵少康。 1月18日，以少壮派赵少康为首的39名国民党立委突然联署公开发表声明，要求推李登辉出任代理主席，“以维护全民信心，以承总统遗志”，同时，改革派的联合报和中国时报马上跟进，报纸上铺天盖地的拥李，声称“李登辉不接任国民党主席，则无法继承改革事业”。 由于这股风浪太为强大，俞国华也一时改变了立场，似乎转为拥李，于是国民党秘书长李焕，在1月24号将拥李风潮告知宋美龄，并且告诉宋美龄，中常委准备开会，提李登辉为代主席。 第二天，宋美龄在报告里回复，说：“根据中国国民党党章，党的主席，常委会无权决定！” 意思再明显不过了，宋美龄反对。而且很快，这个消息就被有心人放了出去。 这样一来，事情似乎朝着不利于李登辉的发展，27号按原定计划召开的中常会，会前，李焕曾经问俞国华，“提不提？” 俞国华不置可否。此时，连李焕都认为，李登辉怕是当不了主席了。 1月27日上午8点，国民党中常会准时召开。此次会议由余纪忠(《中国时报》董事长)轮值主席，除严家淦、谷正纲因病，李登辉请假，宋长志驻节巴拿马，其余 27名中常委均出席会议。各种杂碎事务一项项被讨论，俞国华神色矜持，李焕就像睡着了一般。而拥李中常委也显得焦躁不安。 在关键时刻，列席会议的国民党中央副秘书长宋楚瑜对会议迟迟不讨论李登辉代理主席案，终于按捺不住，决定不顾一切，突然举手要求发言。需要注明的是，宋楚瑜乃是副秘书长，仅仅是列席人员，按道理，他无权说话，如果主持会议的余某人直接不让他发言，历史也就改变了。 可惜，余纪中还是礼节性让宋楚瑜说两句。 宋楚瑜自然当仁不让，第一句话就让满座皆惊。 宋楚瑜说：“今天常会，一定要定下我党主席人选，俞常务委员国华同志，态度暧昧，摇摆不定，这就是没有魄力，没有革命精神的体现！” 如果此时，党国大佬们，哪怕有一个人出来弹压宋楚瑜，事情还有的转机，可在历史时刻，这些人选择了让宋楚瑜表演。 宋楚瑜接着说：“推举代理党主席之议题，多迁延一天，就是多对不起经国先生一天，外间联想揣测必然不能止息，就多有害于党国事业一天，对在国丧中的国家与社会恐将造成莫大伤害，楚瑜对俞同志，以及在座其他同志感到无比失望，也对会议非常不满意，并严重抗议。” 说罢，宋楚瑜居然当场落泪，然后愤而离席。 在错愕一阵后，俞国华站起来，以会议主席的身份，提名李登辉，为国民党代主席。 参谋总长郝柏村，第一个表态支持。其他人也纷纷表态，于是，李登辉成为国民党代主席，一举拿到了党机器。 恐怕，这些人都没想到，拥李的人到了最后，一个个与李登辉决裂，而且有意思的是，当时拥李比较卖力的，还多是外省人，比如宋楚瑜，赵少康，而态度暧昧的则是本省的林洋港，邱创焕，如果早知如此，他们会不会去做当初？ 1988年7月8号，原本对李登辉担任“党主席”有疑虑的宋美龄身穿白底黑纹的旗袍，代表蒋家出席国民党第十三次全国党代表大会，神色轻松的宋美龄只说了一句话：“各位同志，你们好吧”，说完了这句话，宋美龄就走出了大会，从此再也没有回到这个舞台。 从此，台湾进入了长达12年的李登辉时代。 （未完待续）
