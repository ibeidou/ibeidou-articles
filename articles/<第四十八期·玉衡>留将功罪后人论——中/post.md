# 留将功罪后人论——中

**编者按：百年之内无历史。在中国这样的怪现象使我们在浩繁而复杂的事件和人物中只看到了片面和苍白，国民的普及教育使历史人物脸谱化平面化，为尊者讳，虽不执一字于评断是非，却好恶立辨——多年后温习张黎导演的《走向共和》，虽是电视剧作品，却风格凝重，笔法从容，为晚清风云人物填充骨肉，做翻案文章，是中国影视作品中堪称先声的精品。虽仍赋予了文艺家个人的情感，一时间多少人才为李鸿章生于末世运偏消的才干扼腕，才为袁世凯乱世枭杰的气概惊叹，才叹息慈禧的精明独断，有雷霆之风却无远见之能——但推翻了文艺作品和教科书多年的书写历史的手笔，其中所有小节，未必都可奉为史实，但在一时让我们反思对待历史的态度，丰满以往平面的历史形象，一个时代跃然于眼前。而最难得的是将晚清至近代多种思潮一一摆在我们面前，这样一条寻寻觅觅荆棘满地的曲折而困惑的共和之路，满写着中国士人知识分子的责任感和上下求索的赤诚之心——他们仍然在路上。 **

#  留将功罪后人论——中 

## 文/玉衡责编 

剧中谭嗣同就义时的诗是这样说的：手执殴刀仰天笑，留将功罪后人论。我们用传统道德，意识形态衡量功过的历史观，受到了挑战，这句诗也应送给所有在这个时代风起云涌纵横捭阖的他们。  [![1](/legacy/2009/07/14.jpg)](/legacy/2009/07/14.jpg)  从戊戌政变失败始，光绪皇帝被软禁，慈禧的所谓颐养天年的意愿也再未提起。以德国公使在北京被杀为导火索，清朝正式向八国联军宣战。 

片中将慈禧宣战的过程娓娓道来，水到渠成，其中暗流与明线交织；暗流是八国联军的照会触及到了慈禧的底线：要求皇帝亲政，慈禧退位，这是慈禧心中不能触碰的痛脚，触及多年来清廷最深的利益矛盾体。 正在此时义和团起事，同从古至今的农民起义一样，因生计不聊而起，以朝廷招抚或改朝换代而终——清廷将对抗外侮的希望寄托于这些失去了灵魂的血肉之躯，加上朝中如刚毅者再三挑拨，如徐桐者一味愚忠，大沽口战役谎报军情，如此明暗交错，终于促成了向八国联军开战——一日开战，八国联军势如破竹，不在话下。

![3](/legacy/2009/07/31.jpg)

在朝野喧嚣，宣布开战的同时，李鸿章与张之洞却一致咬定开战是伪诏，不予理会，却在同时积极同各国领事商谈东南互保议案，以保住东南经济命脉，更为了开战之后不至于举国沦陷，内外失援，却又先派辜鸿铭与英国公使表明姿态，既逞威武，又谈合约，这让年轻的张謇难以理解——盛宣怀向他解释了他们的曲折逶迤的做事方式，令张謇感叹在清朝做事需要经历太多的波折，动太多的脑筋。而致力于实务的他们为中国的安定和进取所付出的实在难以想象。

[![T_@AW87YI8$FZ5UVJE3$Z[K](/legacy/2009/07/T_@AW87YI8FZ5UVJE3ZK1.jpg)](/legacy/2009/07/T_@AW87YI8FZ5UVJE3ZK1.jpg) 慈禧率众西逃，光绪在威压下不得不随行，珍妃这个从来不能入慈禧眼睛的女子终于被迫害致死，香魄一缕散尽，只留下一口胭脂井，无声无息，这并不只是有关一个女子悲凉命运的遗迹，它还饱含了一个时代的痛楚。让我想到高阳先生的《胭脂井》，笔锋凝重，引人哀叹。从此光绪从当初的曲意顺从到今日的心灰意冷，再无峥嵘之心，大清皇帝名存实亡；他虽有革新之意，却无果断刚毅的个性，这也是他同裕仁天皇在君主立宪之事上相比的先天不足吧。 [![4](/legacy/2009/07/44.jpg)](/legacy/2009/07/44.jpg) 八国联军轻取北京，疯狂的掠夺，而他们对这古老而腐朽的清国又充满了好奇，他们检查了北京青壮年的身体，都是无恙的，然而这个国家的病，不在躯体。 八国联军意欲杀慈禧以及宣战的大臣作为了断，出逃的慈禧与荣禄的争执颇有意思，在荣禄被流放的二十年来，慈禧是心怀愧疚的，他们之间微妙的情感和慈禧穷途末路的境地让荣禄在此时毫无畏惧的说了实话，他知道当初开战时劝阻定然无效，这一切无可避免，而慈禧知道已到穷途，对荣禄任性撒痴露出心内真实的恐惧和懊悔。 在李鸿章的斡旋之下慈禧的命保住了，而那些无论何种姿态，是猥琐，是英勇，是上下撺掇，是尽忠报国，都要为开战负责的大臣却不免一死，而他们的死也带着腐朽溃烂的气息。 [![5](/legacy/2009/07/51.jpg)](/legacy/2009/07/51.jpg) 在李鸿章与八国公使谈判之前，梁启超拜访李鸿章并说出了他的想法，然而李鸿章婉拒了他独立两广，北上勤王的邀请，执意只身赴京签署那万人唾骂的条约——一代人只能干一代人的事，李鸿章这样说道，他在清廷的腐朽和风云诡谲中为洋务，为中国的富强用尽了毕生的精力，也用尽了各种手段，当清廷的宿命走到末路，他仍旧淡然处之，献上最后一份老朽之薄力，将千古骂名留在了辛丑条约之上——李鸿章询问美国上将美国建国年份时，说：真年轻。美国的崛起恰恰因为它年轻，中国的衰落也因为他的古老和陈腐。 是时候了，一个时代即将落幕，如曾，李，张，他们留下了光彩而落寞的背影，他们无力医治腐朽，却一生修补疮痍。 幸而有梁启超为李鸿章做传，再翻开这本书，对照此剧，不禁感慨怅然。 

[![6](/legacy/2009/07/61.jpg)](/legacy/2009/07/61.jpg)

 袁世凯的时代终于到来，慈禧逃亡之后他令杨杏城征得十几处行宫，到天津为慈禧的驾临准备警察以保安全，到直隶火车站接驾，所有的一切都令慈禧大为满意，袁世凯知道，这中国仍然还是慈禧的天下。杨杏城令袁世凯颇为赏识，因为他识时务，会机变，深深的了解中国官僚的弱点和官场的潜规则，而这侧面写出了在当时若想成事不得不屈从于这些规则，而诺大的中国官场，竟无一例外。 

[![7](/legacy/2009/07/74.jpg)](/legacy/2009/07/74.jpg)

 康有为和孙文为普及他们的思想，为改革或革命募捐，在国外各自搭台唱戏，相互拆台，康有为仍旧惦念着戊戌年光绪皇帝的恩情，一心想要为皇帝夺取宪政，而孙文倡导国民革命，彻底推翻帝制，你方唱罢我登场，如火如荼，皆有人响应。 

[![8](/legacy/2009/07/81.jpg)](/legacy/2009/07/81.jpg)

袁世凯新政之始便是取消科举，建立新学。清政府的人才机制腐朽，刻板，科举制度是几千年来的沉疴，而以辛丑条约中签订的义和团起事地区生员禁止参加科考为契机，干脆就彻底废除科举，可怜万千只会读圣贤书做八股文章的举子只落得一片凄零。科举正是袁世凯仕途曲折坎坷的主要原因，他在朝堂上与瞿鸿畿等人辩论，又未妻妾请新学教师，令她们为新学学堂争得正当权益，正是志得意满，吐气扬眉。 

[![9](/legacy/2009/07/91.jpg)](/legacy/2009/07/91.jpg)

杨度在日本遇到军国主义狂徒，受到屈辱，忿忿不平，借酒浇愁，与孙文二人醉倒于街上。作为多年研究立宪的专家，杨度咬定中国之革命，皆如陈胜，吴广，太平天国，义和团，乌合之众不能成事，不过改朝换代另立君主，是历史的循环往复；孙文则咬定清王朝腐朽不堪，不能再寄予期望，他们互骂却又相知，其实他们都知道，未来是未知数，杨度与孙文约定如果立宪失败便加入共和的队伍——回顾历史，立宪失败了，而共和是否确实成功？杨度当年的担忧是否成真？这场争辩虽然以烂醉告终，然而留下来的问题却引人深思，对照历史，不得不从新将当初二人的对话反复深思。 

[![10](/legacy/2009/07/103.jpg)](/legacy/2009/07/103.jpg)

再道国内，世界大潮浩浩荡荡，慈禧集团终于露出了立宪的意愿。这是一个全新的世界，却也是一个不得不顺应的世界，再三权衡之后，慈禧集团决定派五大臣出洋访问，考察各国政体，以备立宪。 而刚刚准备出行的五大臣就被孙文的同盟会炸车，出师不利，这使清廷痛心疾首，更加仇恨革命党。 

[![11](/legacy/2009/07/113.jpg)](/legacy/2009/07/113.jpg)

片中写道五大臣赴美国和日本的考察，一为共和国，一为立宪国。由罗斯福和伊藤博文向我们道来，共和与立宪的区别，宪政之要义。 在美国，民治，民享，民有的三民主义是孙文为了在美国宣传他的三民主义所翻译来的，共和宪政的要义即三权分立，有监督制约机制，服务型政府等，或者美国宪政的成功也有赖于它的年轻，共和之朝气薄发，中国以冗重之躯，何以效仿？ 

在日本，伊藤博文对中国充满了仇恨，他翻开梁启超写的李鸿章传，竟潸然泪下，李鸿章胜于伊藤，却败在生在专制之国，以一人之力担负举国之沉疴，伊藤不才，生在宪政之日本，却胜过了李鸿章——伊藤博文是惋惜的，更是自豪的。 [![12](/legacy/2009/07/123.jpg)](/legacy/2009/07/123.jpg)

 在一番考察之后，清廷决定顺应潮流预备立宪，其实中国知识分子早已对宪政研究的无比透彻，袁世凯请来了杨度，而讽刺的是，被清政府驱逐到境外的梁启超的著作最终也被用来做立宪参考读物。 预备立宪宣布开始，为期十年，而孙文咬定了这是慈禧的权宜之计，假立宪而为利益集团谋私——虽此时片子对慈禧集团的真正立场中不置一词，只观日后之效。 

[![13](/legacy/2009/07/131.jpg)](/legacy/2009/07/131.jpg)

 果然在立宪伊始，就触动了各方面的利益，从王公贵胄，到军政大臣，精简人员，狠下裁刀，真正的宪政触及的并不仅仅是一个人的利益，而是整个沉冗腐朽的既得利益集团百年不变的利益。 庆王爷奕诓和袁世凯执掌者宪政大权，几位大臣想合力留下利益的种子，便向庆王府蜂拥而去，然而立宪是什么样的大事？奕诓知道，虽然贪婪却还是慎之又慎。瞿鸿畿断定这是庆王爷和袁世凯专权的一个借口，他并不与那些大臣同流，却意图从道德上参倒他们。 

[![14](/legacy/2009/07/141.jpg)](/legacy/2009/07/141.jpg)

宪政之初，废太监，去礼仪，从小事起，便受到阻挠，以国情不同为理由，便可独立于世界文明，遗留糟粕，终是难以打破几千年来国民性和文化的桎梏。李莲英仍旧主持为小太监净身，还告诉他前途大好，那腐朽渊源绵长，一眼望不到头，当大臣们议定废除礼仪之后，慈禧仍要对身后的奴才们说：你们怎么还跪着呀？ 

[![15](/legacy/2009/07/15.jpg)](/legacy/2009/07/15.jpg)

新官制是改革的中心，更是触动利益团体的最重要的方面，在王公贵族面前袁世凯不动声色，露出枭雄之姿，王爷们，大臣们甚至不惜拔枪震慑，围殴动粗，终是无力抵抗袁世凯锐意兴革的决心和独揽政权的野心。面对非议和职责，袁世凯显露出的是舍我其谁的气焰，立宪不得不说是一个人的立宪，慈禧再极力抓取自己在的几年中的权利，都已经难逃气数将近的命运，无论如何这都将是一个袁世凯的时代。
