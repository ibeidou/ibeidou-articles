# 从一个德国人口中的Nation-States 说起

## 文/[谢熊猫](http://panda.n2studio.org/)

学校的老师中有一个来自德国的年轻人，大概30 岁左右，英文讲得很好，在人文学院教书，也教德语，上学期他教过我一门课。这学期我学德语课，可惜并不在他的班上，和他讲起时俩人都觉得颇为可惜。上学期上他的课时我的表现不错，在课上时常讲些笑话给他听，并且还小小的就scientology 讨论过一下，颇为谈得来。平时在学校里时而能碰到他，每每谈一些轻松的话题。因为都是年轻人，倒也没有师生间的隔阂。与他交谈时我最先意识到的一个特点是，虽然我和他是用英文交流，而且英文于我俩人都是第一外语，但是谈到“国家”时我们的叫法很不一样。我习惯把“国家”叫做countries ，而这个老师习惯用nation-states 。其实nation-states 这个叫法并没有什么不妥，理解上完全没有问题，只是完全不属于我以及我周围人所用的词汇范围。后来稍微一研究发现其中还是有点意思的。 其实在英文语境中关于“国家”的表述有好几种，比如nation, state, country 还有先前提到的nation-state ，要理解其中的微妙不同之处单靠背单词解释是不行的，更要从平时的阅读写作中潜移默化地来了解。常读报纸杂志的人都知道讲到西方国家必然是说Western country 或者Western nation ，绝不会用到Western state 的，而说到伊斯兰国家的时候则常用Islamic state, 却不用Islamic nation 。这种选择性的用词其实不只是简单的约定俗成，更是有其社会语言学因素的。而这其中的差别最根本就在于族裔民族主义（ethnic nationalism ）。 族裔民族主义特立于欧洲大陆，除了民族主义的民族文化特征产生的自我认知这一特点之外，还包括国家与居住地这两个概念在地理上的基本吻合。族裔民族主义的一个最直接的后果就是民族国家亦即前文提到的nation-state 这一概念的诞生。因为在欧洲人的观念里，民族国家（ethnic nation ）和政治国家（political state ）在地理上是基本等同的。事实上在西方文化的语境里民族国家和国家几乎是同义词。民族国家的这种认知也是西方语境中的“公民”这一概念的必备条件。但是伊斯兰教国家就不是这样了，虽然像埃及、伊朗这些国家也有自己辉煌的历史，但是这种辉煌的过去是建立在考古发现上的，而不在于传播下来的思想之上，所以他们并没有自己可以追溯的民族历史来为他们的nation-state 的概念创造条件。至于其它伊斯兰教国家则更加不济，它们多是在西方的压力和影响下成为现在我们所知的国家的，对于这些人来说民族和国家共存的这种认知感是很难建立的。试想那些生活在大陆中间的山区的阿尔及利亚人怎么能把自己的身份和algeria （意为“岛的”）联系到一起？ 公民这一概念的基本原则是一个对于国家尽责的个人，通过他对于国家的忠诚来获得自己所期望的利益。在西方思想史上“公民”这种概念经过了长久剧烈的演变，成为公民的标准也一直在变化，从西塞罗时的荣耀（honorable ）到洛克的有生产能力（productive ）。到了现代，在绝大多数国家公民身份都是在出身时便被赋予了的，这种变革最先自然是始于西方，而后逐渐通过西方帝国主义的殖民传播开去。这种影响甚至在很多伊斯兰教国家也是有的。然而可惜的是传播开去的只是这种形式，而并非“公民”这个概念所包含的深层意义。 “公民”这个概念所包含的一个最基本价值就是个人通过对公共美好（common betterment ）的贡献来达到自我实现。而实现这个价值的基本环境就是公民必须有这种自由去做贡献，并且想去做贡献。这就牵扯到一个西方人常见的误读。西方的整个流行文化和教育体制都有意无意地把一些价值观和思想自然而然地当作所有人都追求的，即所谓的“普世”。自由、权利、公民社会、宪政等等，不管是从含义还是外延上来看都是好东西，但是如果真的要放之四海皆准就很难了。试想一下，这些以公民概念为基础的价值，在一个缺乏公民概念而且缺乏真正理解公民价值的人的地方会是被大众所推崇的吗？事实上在伊斯兰教国家，公民这个概念在奥托曼帝国解体之前都是不存在的，公民这一概念是随着西方体制的引入才逐渐出现在当地语境中的。但是仅仅引进一个新兴的概念是否足够呢？显然不行，如前文所说这些体制和价值并不是这些人们在历史的斗争中自己创造发现的，而是在西方的压力下引进的，所以这些地方缺乏对于这些概念的广泛认知，在他们的文化中没有古希腊城邦这些能起到基本象征作用的元素。于是没有历史背景的公民概念就好像半路出家的和尚一样，只是虚有其表。 做一个简单的类比，就好像有些地方的人以昆虫为美食，食用昆虫不但有营养而且有口福，一切都是天经地义。但如果强行把吃昆虫这种习俗传播给一个从没有吃昆虫的历史和习惯的地方，尝鲜者自然会有，但是要对整体饮食文化产生大的影响怕是不可能。这些西方的价值观在伊斯兰教国家不讨好，然而很多人却不是从文化或是公民价值的角度去考虑，反而把责任推到了政府的威权统治上面。但是美国人在伊拉克的实践却狠狠地给了这种说法一个耳光。美国大兵推翻了伊拉克的暴政，把自己的体制和价值观介绍给了不再受到威权统治的伊拉克人民，结果还不是吃力不讨好？其实究其原因就在于西方的体制和价值给了普通人来为集体的繁荣做贡献的自由和能力，但是意愿这种东西却是强迫不来的。没有公民价值大传统，又怎能有这种把奉献当作自我实现的意愿呢。 再回到开头的那个德国人，对他来说把国家说成nation-states 可能像我把国家说成countries 一样，他的理解更多在于民族国家和政治国土的对等，我的理解更多在于政治体的土地。这中不同的理解对于我们来说都是理所当然，所以这并不是一个关于对错的问题。很多问题都没有对错，因为所有人都认为自己是对的。如果把这种不同的理解放大来看又怎样呢？奥巴马的就职演讲惹来不少讨论，或褒或贬，但是至少有一点他传达的比布什总统好。布什当政时关于对外战争的表态把世界上的国家分成两种，一种是支持自由的，一种是反对自由的。在布什看来，打仗的目的就是把自由带给这些被反对自由的国家所压迫的人。且不说这种政治说辞是否另有意图，这种出发点的傲慢和对于对错的定性就过于理想化了。伊斯兰教国家经历的是和西方国家不同的发展历史，但是不同的历史并不代表其文化是错误的或者不好的。奥巴马在这点上面的表态就要谨慎许多，他诉说的是一种建立在双方利益和彼此尊重上的新途径(based on mutual interest and mutual respect) 。 话说回来，之前曾和友人在网上讨论到语言是误读的罪魁祸首这个话题，友人的观点是语言是文化误读最主要的原因。我则认为并不只是语言这么简单，就如同公民这一概念，即使都是用citizen 这个词来表述，其真正含义也是大相径庭纵深有差的。本文除了浅谈一下我对西方价值和制度在伊斯兰教国家的适用性之余，也算是给那次没有完成的讨论做点补充吧。
