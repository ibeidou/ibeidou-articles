# 追寻真相，捍卫人道之三——战斗的三人

# 追寻真相，捍卫人道（三）

# ——一个记者对当代中国的观察与思考

## 王克勤

### 第三部分 战斗的三人

### ——疫苗事件之后的明争暗战与来自网络的合力

 最有意思的事情是报道签发之后出现了一系列有趣的博弈和冲突。报道发表后，山西省卫生厅当下下午紧接着就通过新华社发布了消息，当天下午新华社还发布消息称，山西省省委省政府认为这个报道“基本不实”。（众笑）第二天下午18点，我和我的总编辑包月阳、副总编辑车海刚在一起碰头，我们讨论来讨论去最后认为，对于这件事情我们不能保持沉默，于是就推出了在中国新闻界反响非常大的《关于山西疫苗乱象调查报道的声明》。现在我可以把当时的细节讲一讲。稿件发表之后，有人称我们是“三人自由小组”，（众笑）怎么说都行，凡是当时我们三个人大概每过一两个小时就会碰头一次，随时把各自得到的信息汇总到一起，如同作战指挥部。我们跟谁作战？跟山西政府作战，跟卫生部作战，跟新华社作战，我们面对三大劲敌，三线作战，如何打？（掌声）按照“惯例”，面对如此强大的压力，一般的报社就把自己的记者“卖”了，然后报社的老总就开始写检查。（众笑）所以凤凰卫视的杨锦麟说，我们的这个声明“掷地有声”，为中国的新闻人赢得了光荣。我们对事实负责，对每一个人的生命负责，这是我们的本色。报道出来之后，卫生部和山西省政府跟我们闹腾，我们怎么办？当然是要写一个声明。一开始，总编辑包月阳跟我说，克勤，你写一个博客，驳斥他们。后来我们副总编辑说，你写一个博客只是代表你个人啊。因为当时我的博客点击量非常高，N多人都在点击我的博客。所以就决定以报社的名义起草一个声明。我就开始马上起草。起草完了总编辑给我说，你上来！于是我就腾腾腾跑上去。总编辑看完声明，说：“你这个声明，力度不够！”他抓起我这个声明，改了几句，然后说：“一定要强硬！”我说：“对，强硬！”（众大笑）一个很强硬的声明就出去了，我们就这样强硬地挑战了三大集团。 发表声明之后，我们一直摸不清新华社将对我们怎么办，（众笑）摸不清。我再给大家解密一个秘密。稿件发表之后，我的办公室电话超级火爆，用“火爆”这个词一点都不过分，很多北外的同学也在那里接电话都接傻了。（众笑）国内外媒体电话很多，有自称CCTV的记者从山西打来电话。干啥？刺探情报。对他们而言，我是敌方，他们要打探战况。（3月）17号下午，我接到新华社山西分社的一个女记者打来的电话，这跟记者对我说，我对你很佩服，因为我经常说这样的话，所以我说谢谢你对我的认可，之后我们就开始交流，因为来电还有很多，我就跟她说，我们下次再交流。结果之后，她又打来了第二个、第三个电话，从话音里我就听出，来者不善，跟我在下套。这个人的名字我就不说了，因为她还年轻，后来有网友从网上人肉搜索出她的照片，发给我我也看了，很年轻，我觉得她人生还很漫长，我就跟朋友说，别再人肉搜索了，年轻的时候，谁不犯一两次的错误，谁没有一两次闪失呢？所以当时我认为新华社在下套，整个对新华社报的是一种很失望的态度。 山西分社可能被山西省卫生厅或者被山西的煤老板们用钱砸晕了，那么新华社总社为什么还会发把《还原“山西疫苗事件”的真相》这样一篇东西发出来，甚至还写上了“疫苗真相”这样的字眼呢？《还原“山西疫苗事件”的真相》这篇文章居然不敢署名，署名是“新华社记者”（众笑），这人没名字吗？就没有具体的人对这篇稿件负责任。后来，山西疫苗患儿家长们去新华社山西分社门口请愿，分社的领导最后把他们请进去，承认是他们的记者作了假，是卫生厅的人要求他们以他们记者的名义签发。那么为什么他们能够答应人家作假的新闻在新华社的整个发稿渠道中流通并发布呢？是不是山西省卫生厅砸了很多的钱把他们砸晕了？这个咱们不得而知。我希望在座的同学们能去调查，特别是咱们《107调查》（北外新闻系系刊，本次讲座的主办方），有很强大的调查团队，也许能调查出一些很有意思的内幕。 接着呢，到了19号的时候，山西方面来了一支队伍，傍晚时分悄悄进村，到了报社，与我们报社进行了首次官方接触，在夜色下。（众笑）这次接触很有意思，先是赔笑脸，然后是点头哈腰，说了一大堆，最后山西省卫生厅副厅长也来了，这个李书凯在那个场合还在撒谎。我一直不想说话，后来我觉得我忍不住了，我说，还在说谎，太可怕了这个人，无药可救了。我就地就端出来几条，我说你给我交代咋回事，结果当场就叫我给堵回去了，最后因为当时在座的人很多，两军对垒，进行谈判吗，坐到最后脸色煞白，（众笑）后来就什么都不讲了，开始瞎说什么“我们从网上看到王老师你有多么多么多么……”，总之是极尽拍马之能事。当晚他们力邀我们报社总编辑吃饭，我说因为他们来的是山西省卫生厅副厅长、山西省宣传部副部长，我说他们不够格，总编辑不要接待他们，我们接待！我们有副总编，因为我们报社规格也比较高，我们接待就可以了。（众笑）接待完了之后，双方谈判完毕，我们总编过来说，请山西来的同志们到我们本报的食堂吃个饭，然后在本报的食堂，一人一个份饭，（众笑）然后在热烈的气氛中、在沟通的气氛中宾主结束了晚宴。吃饭出来之后就有精彩的故事发生了。厅长、部长跟我们副总编在前面走，具体的我就不能讲名字了，否则这些同志就没法在山西继续工作了。总共来的还有两个三个处长，他们在后面偷偷拉住我：“好！好！”我都不知道在说什么。（众大笑，掌声）另一个处长悄悄说：“我是迫不得已，被抓差抓来的，身不由己啊！”他也良心不安啊。我觉得人心都是肉长的，每一个人心中都是有杆秤的。虽然山西当局对我们的报道呈打压之势，但，人，普天下的老百姓，心中都有一杆秤。（叹气，众鼓掌） 到了21号，中午12:57，又爆出一个“砍腿门”。发来短信给举报人陈涛安，给他家里发来短信，说：“如果你执意要闹，我们老板花钱找人看你一条腿，还是很容易的，你的地址我已经记下来了，不要以为闹了大事就有用，事情一过你还是普通人一个，我们老板可不是普通人，真要搞你，你就是告到哪里都没用，希望你好好考虑一下，另外不要试图知道我是谁，我也是受人之托。”活脱脱一个黑社会嘴脸，我都不知道说什么。不仅是陈涛安，当时在太原的几乎每一个患儿家长都收到了类似的恐吓短信，这已经是第二个短信。第一个是说，如果你不在告状，我将给你五万或十万。第一条是收买，第二条是砍腿。收买、利诱、恐吓都有。 然后到了22号，砍腿门之后的第二天，山西省政府召开了新闻发布会，只开了10分钟，草草结束。什么宣传部的副部长、办公室的主任被好多记者围起来，就赶紧跑掉了，好几个记者当场拍桌子：哪里有这样开新闻发布会的？不让人提问。你们以后要干记者的要注意，以后经常要面临中国的这种要快速逃窜的官员。（众笑） 24好晚上七点钟，中宣部新闻局下达禁令。所有在山西采访的记者全面召回。我们经常接到召回令。后来24号晚上，山西警察进北京“接”患儿家长王明亮。王明亮到北京陪自己要住院的哥哥看病。于是我就把王明亮接回我家了，我说，只要我在，就休想把你抓走，然后就把他带回我家。之后我的总编从博客中看到我发布了这条消息，消息发布之后呢，天下的网友就打电话给警察把警察打急了，警察的老婆的电话号码居然也给搞到了。警察所在的乡里的派出所的所长就立刻下令，所有警察全面归退。所以我今天晚上又要发布博客。高长宏现在正在派出所情况不明，我的学生现在正在我的博客上发布这个派出所的电话和交口县公安局的电话，希望同学们踊跃拨打！（众大笑，掌声）这是网络时代、信息时代公民维权的一种模式，用网络联起来我们大家共同的心和力量，形成一种合力。2005年我在湖北巴东县采访的时候被困在一个山头，当时就有很多媒体通过饭否了解到情况，就给巴东县公安局打电话，到后来巴东县公安局一听到普通话就说：“王克勤目前一切情况良好！”（众大笑）所以当我们坐在我们的教室里、宿舍里、任何一个能上网的地方时，我们每一个同学都能对推动社会进步付出自己的力量！（掌声）网络是一个很有效的方式。我也是网络维权的一个受益者，我曾经被网络、被网民们保护着，所以我也经常会感谢。连家宝总理、锦涛书记也是网民呀，从他们上网的那一刻起他们就是网民了。 26号，卫生部发布了一个“个人不得发布相关消息”的通报，接着呢30号，江苏常州、河北、辽宁、江西、广东、湖南等地发现问题疫苗，直到4月6日卫生部召开新闻发布会。当卫生部的新闻发布会，中青报的记者刘万永批它是“臭不可闻”。刘万永在自己的博客上批了很多，最后卫生部发言人给他打电话，问：“你能不能再不批？”（众笑）刘万永回答说：“我可以把你这些话也写上去。”结果卫生部再也不敢给他打电话了。这也是网络时代网络被我们带来的好处。 实际上3月17号之后，我的办公室自动成为中国疫苗救援中心，直到4月30号为止，我的办公室收到来自全国各地有准确详实证据资料的投诉共305例，总共死亡18人，山西159例，死亡12人，全国其他地区146例，死亡6人，问题疫苗涉及全国24个省。我的感觉就是，无处不被问题疫苗。 说到这里，有一个重要的资料想给大家呈现一下。当山西有关部门和卫生部凶猛反扑的时候，我们搜到了中国疫苗的最高业务主管单位——中国CDC即中国疾控中心的行政长官王宇，在今年两会期间的一段讲话。王宇在讲话中是这样说的：“部分国产疫苗不达标。”王宇作为全国政协委员，在政协会议医疗卫生界分组讨论上讲：“很多国产疫苗在群体性接触过程中存在不安全因素，国产疫苗与进口疫苗之间的差距很大，提请卫生部对目前已经接种的群体做全面的普查。”咱们做个形象的比喻，儿子跑到广场上说，爹爹怎么不管事儿，却不在家里说。由此可以说明，中国疾控中心主任王宇曾经无数次的给卫生部汇报过中国疫苗存在大量的问题，但被置之不理。这是我看到东方卫视关于两会报道的一个短片，有电视的，这是里面王宇的原话，否则我不敢信口雌黄。我从王宇的整段话中可以感觉到，整个中国的疫苗确实存在深深的隐患，我只能这样讲，因为有关部门不准发布信息，我也不敢发布信息，我也不是发布疫苗有关信息的主体资格人，我是不过是记者，我看到了我听到了，说出来罢了。 所以好多人问我，最近全国各地的疫苗都出事儿了，我的孩子要不要接种疫苗？对于这个问题，我确实不好回答。但是我一直坚持两点：第一，要坚定不移地接种疫苗，这是我一定坚持的；第二，如果你家里比较有钱或者特别有钱，我个人建议你接种进口疫苗，比较安全有效一些，我只能这样说。当然我希望我们的国产疫苗愈来愈有保障，让更多孩子的生命愈来愈安全。 本来有关中国疫苗安全的问题，有很多人很多学者都提议召开一个中国疫苗的研讨会。然而不幸的是，就在研讨会的前一天，我被管制了，我被打压了，所以这个会议就取消了。当然，管制的力量是很强大的。我在办公室里跟我的学生，把手机电池去掉，把Sim卡片去掉，把所有带电的系统全部处理之后，我们的谈话还是被有关部门知道。所以同学们以后要做揭黑记者的话，应该有这样的基本素养，当然这样的工作做起来确实很刺激，但是也很艰辛，而且更加危险。要知道，就在五天前，我的总编辑被免职。疫苗的事情我说到这里，谢谢。
