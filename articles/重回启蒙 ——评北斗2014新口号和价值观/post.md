# 重回启蒙 ——评北斗2014新口号和价值观

首先在我看来，这个“重回启蒙”的方向非常正确，但是仍有三点我想着重提一下。

1. 精英预设

第一是斗人对知识精英姿态的认知。精英姿态其实并不是一种姿态，它也许只是一种客观现实，而这种现实，其实并不应该放在考虑范围之内。拿北斗最初的的理念和所作的事情来说，实际上就是由启蒙程度深一点的青年，去点拨启蒙程度弱一点的其他青年。如果更多的知识和更深入的思考与表达可以划归为精英的产品，那么虽然第一代斗人和撰稿人很可能没有精英意识，但他们做的确实是精英的事情。无论斗人是否有精英的姿态，我都认为这样解读“致力于推动青年人自我启蒙”是错误的，即”预设了自我话语权，好像是对受众说，我来帮你启蒙，你没有这个能力”。

因为，北斗的启蒙，从来都是加深启蒙，而非开始启蒙。启蒙的起点一定是自我的，北斗这样一个网络平台并没有使受众从无知与虚假枷锁挣脱出来的能力。我们应该已经有了这样一个认知，能被北斗推动自我启蒙的青年，其实已经因为各种好运迈过了自我启蒙的第一个台阶。

另一方面，到底怎样才能使人们加深自我启蒙的过程？如果按照康德的解释，“公众要启蒙自己，只要允许他们自由，而且只需要无害的自由，那就是在一切事情上都有公开运用自己理性的自由“，其实说的就是一种理想化的言论自由。假如接受康德的这种解释，那么只要有言论自由的空间，那青年人就能自然而然的自我启蒙。但问题是我们并没有这样的言论空间，人人网上可能有一些，但无疑也是权力压迫下有点扭曲的言论空间。

那么在康德所讲的自我启蒙的前提不完全存在的情况下，北斗最初到底是如何“推动青年人的自我启蒙”的？这就回到了所谓对知识精英的认知。那就是在自我启蒙或加深自我启蒙可能难度很大的情况下，斗人及撰稿人作为先行者，以更丰富的知识，更深入的思考，在有限自由下更巧妙的表达，来推动读者的进一步自我启蒙。这一套逻辑确实是“因为环境不自由，所以能力受到限制，所以我们来帮你启蒙”，但并不意味着背后蕴含了一种知识精英的姿态和预设了的自我话语权。

也就是说，相较读者，你的知识更多，认识更深，表达更有力，只有这样才能推动他人的进一步自我启蒙，这样的人算不算精英，仅仅是个定义问题罢了。

但是，若从“致力于推动青年人自我启蒙”变为“北斗，青年自我启蒙践行者”，也的确意味着北斗的形象发生了变化，暗示着斗人和撰稿人的自我启蒙程度可能是跟读者不相上下，并没有作领跑者的意思。同时，北斗的身份，也就从推动者变为了服务者，看似是应对网络时代环境改变后自我转型之举，但本质上是因为丧失了做领跑者的能力后无奈地放低姿态。从组织活动和建立线下联系与交流这方面，北斗的服务确实很重要。但在内容上读者需不需要和自己知识水平与精神气质差不多的服务者，就有点难说了。


2. 重回启蒙

如果将口号重新定义为启蒙，那么到底什么是启蒙，如何让读者明白这句口号的涵义就变成非常重要的问题。至少在我看来，“启蒙”的意义在现时的舆论领域已经被完全刨空了。说刨空并不是指启蒙已经成为一个负面的词汇，而是说启蒙变得毫无意义。人人网内的舆论圈，已经不太需要再由别人来帮助和服务自我启蒙了，在人人网之外的社会主流舆论，启蒙也早已被解构得一无是处。

我完全赞同“启蒙对当下中国青年人来说仍然是一个核心任务”，但是要能让前来北斗的读者认识到启蒙的价值，看到这句口号后能在他心中留下一丝余音，则是一个非常困难的任务。在舆论领域重新构建启蒙的基本意义之前，一起来号召“青年人自我启蒙！”就像是一个空洞的口号。

所以若在这个时候北斗重新将口号与启蒙挂钩，应当既有北斗本身回归启蒙的认知，也有号召外部舆论重新认识启蒙的意涵，是内外两方面的return to enlightenment. 仅从这些方面来考虑口号，我觉得不妨考虑“回归（重回）青年人的自我启蒙”等说法。

3. 品位要求

我始终认为，在互联网形态发展到可以说是web 3.0的今天，修剪枝叶，控制编辑数量，严把质量关，每天能做到推送一篇原创文章，或是隔一段时间做一期自己的内容例如七星说法等，是符合当今互联网及移动潮流的最好选择，同时也是团队输出和用户接受内容的极限。换句话说，其实就是走“一个”的路子，并且对于北斗来说，最好能禁得住自己产出编辑生产内容的冲动，即使要做，也需要格外的慎重。

要提高内容的品质，首先得提高编辑的品位。除了对文艺的品位，更重要的是对政治的品位，对人文知识的品位。而且后两者才是北斗区别于其他很多网络团队的最大特征。不管怎样，编辑的选稿品位一定要高于一般读者的鉴赏品位。综合来讲，就是如何运用自己的智慧和品位，鉴别出那些最具自由精神的作品。

这就是说，即便是和读者共同启蒙，斗人的自我启蒙程度还是得领先于其受众，或至少是在某种机制下更快地进行自我启蒙。“思想随人走是自然而然的，把它作为集体的记忆遗传下来却是非常难的“，这的确是一个无法摆脱的残酷事实。我在这里又提这个问题好像是在说废话，重申这一点，是因为在我看来，这是北斗能不能自我重塑，重新上路的标志。
