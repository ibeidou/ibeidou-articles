# 汤社志之﹒砖茶



《茶经》上书：“一曰茗、二曰槚、三曰蔎、四曰荈.”

荈指采摘最晚的粗茶，老化的叶片自然不比早春茶的新鲜劲。粗茶一般不会被制成绿茶，它们被摘下后要经杀青，翻炒，焙烤，压制等过程，变成发酵茶，也就是红茶或黑茶，以黑茶居多。黑茶多被压制成砖块型，硬度也堪比城墙。砖茶上不了大雅之堂，对于边境的少数民族来说却是重要的饮品——它可以很好的帮助消化油腻的肉类与奶制品，提供维生素的来源。

最初关于砖茶的记忆，源于在外公家度过的那段日子。

外公在文革时因成分问题被划为地主，当时与他家境相当的人均被逼迫自杀或处决。他在面临被枪毙的危险之前，带着舅舅逃去了内蒙古。后来拨乱反正，外公回到了南方，每天喝砖茶的习惯依旧保留。

不同的茶对水温的要求不一样，碧螺春经不起烫，正山小种就得要沸水浇才能出透香气，而砖茶，乍看之下和石头没什么区别，却是要投入壶中久煮的。茶艺馆里破茶砖用茶针，要截到砖的每一层截面才能保证茶味的完整。当过军人的外公不兴这套，他直接拿起菜刀，对着案板上玄武岩般的物事就是一下，手起刀落，三寸见方一块，正好一大壶茶。

窗外下着雪，外婆戴着顶针，坐在床头给我缝一身红棉袄。炉上的铜壶咕嘟咕嘟的冒着泡，旁边放着我大清早出去疯跑在河边踩湿的鞋，我披着外公的羊皮内衬军大衣，膝盖上放着外公的“红灯”收音机，发出吱吱的电波声，电池盖子早已不见了，露出两节大电池。外公端着搪瓷杯子，倒了一杯，喝一口，再拿了个玻璃缸，将茶汤滤进去，递给我，说：“吹吹再喝，当心烫。”

陇海地区的井水有特殊的盐碱味，煮出来的开水难以下咽。每年在外公家的日子里，只好靠茶度日。外公是个讲究的人，喝茶的时候会从门后的布袋子里掏出两块桃酥，就着刚煮好的茶，黑茶虽然粗，却并不苦涩难咽，那种味道像黑糖一样充满杂质，显得纯朴的很。手里捧一杯，其余的还在炉上静静的温着。外公带上老花镜，看着杂志。一只花猫静静的窝在他脚边。

“当年在内蒙的时候，蒙古包里的人早上都煮这个茶。还得加奶，加盐，加炒米。那味道，得劲。”

“大冬天晚上从一个公社赶到另一个公社去，要骑马过草原。怕被冻死，就喝好多酒暖身子。结果喝太醉了，和别人在路上的时候，我睡着了从马背上摔了下去，那个人吓坏了。马自己认路回了公社，我被骑马路过的人接了回去。”

“和蒙人喝酒一滴都不能剩……我们呆的那个地方叫巴彦淖尔，蒙语里就是富饶湖的意思。内蒙古那些城市在蒙语里都有特别的意思，呼和浩特是青城，包头是鹿城。”

外面的雪一直在下，虽不及外公说的内蒙古的雪下到齐膝，踩上去也能陷下去一大块。外公说“我可得活到看着你上大学”。妈妈是外公外婆快50了才有的孩子，外公一直很怕自己老，每天煮茶，出去溜达，骑自行车。直到后来我随父母举家搬去上海。那座小城，那场雪，那壶茶，仿佛都在我记忆中淡掉了。不知不觉，我上初中，上高中，上大学，外公已经九十岁了。

他试图给我打电话，可是耳聋什么都听不到，他浓重的口音，我时隔多年也很难分辨。直到妈妈将他接到上海，我开门看到他，还带着高顶礼帽和老花镜，外公老了，我却长大了，他已经不能像当年那样两手拎起小小的我，大喊一声：“一二三，走咧！”然后三步两步跨过混着融雪的水洼。他的草原，他的蒙古包，他亲手种的沙棘树都还在，他却再也无力坐上三天三夜的火车回去。我在这座充满人造星光的大城市生根落地，又去另一个地方求学，我也已经回不去了。时间与空气氧化世间万物，以不可逆的方式。

粗茶只是粗茶，平淡的让人忘了放到精致的茶盅里去品。没有成色，没有光圈，只有久煮的温度赋予它的温暖。记忆里的故乡，若非要味觉来唤起的话，便是一块一块的茶砖，组成的一座模糊的城池，到处浸染着茶渍般浅褐色的污渍，水开了，咕嘟咕嘟冒着蒸汽，扑面而来，留下一脸细密的水滴。

“武汉那个地方，走一日走不出武昌，走三日走不到汉阳。”外公坐在沙发上看电视，戴着老花镜。“当年我们行军路过武汉，看到黄鹤楼。武汉被轰炸的稀巴烂，荒无人烟……”

当年十六岁的外公可否想到，他的外孙女在他九十岁时，到他行军路过的一处昔日的废弃城市去上大学。戎马一生的老人，却在晚年，如离了灶的壶中茶叶一般，沸腾后，沉淀下来，满是故事。而最初的茶的诞生，正是源于古代行军打仗时治疗瘟疫与伤口的汤镬煮药草。茶本非稀罕物，一味药而已。或者我该感谢发明茶的人，为几千年后的人制造一个用热水与植物建立起来的温情纽带。

外公在上海呆了几日。他喜欢莉莲的蛋挞，每天走一公里去公园下棋，能记得住10位数的楼道门密码。那天父亲带回一盒特级武夷岩茶，沏了一壶给外公尝尝。

外公说：“这不跟咱家砖茶一样好喝。”

我背过身去，眼眶湿了。

茶出出则香，俗呼离乡草。
