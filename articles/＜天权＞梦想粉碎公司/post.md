# ＜天权＞梦想粉碎公司

**公司的新员工太多了，已经占到世界人口的一半。在这样多的员工的努力下，我们公司的事业也将蒸蒸日上。老板决定，让一部分员工离开公司，回到他们原来的生活里，做回他们的职工，演员，厨师，父亲，兄长和伙伴。这样，渗透了生活的各个角落，有助于他们更好的完成自己的业务，彻底地粉碎掉人们的梦想。**  

# 梦想粉碎公司



## 文/卫京京（香港大学）

 

**一**

我在一个叫做梦想粉碎公司的地方上班。我的老板是个很无聊的人。这一点从这个蛋疼的公司名字就可以体现出来。无聊的人一般有两种。一种是无能的人，比如废柴大叔，废柴大叔，还有废柴大叔，这三种。无能的人想做的每一件事情都不能实现，这让他们感到灰心丧气，便对一切事情失去耐心和兴趣。还有一种是无所不能的人，比如我的老板。他想做的每一件事情他都可以实现，这让他感到痛苦万分，进而对一切事情失去了耐心和兴趣。举个例子，当他看到美丽的夕阳缓缓坠入天际时，被美景感动之余不免有几分遗憾，不由心念“要是这美景能够停留该有多好”，结果夕阳就卡在了天际上。他仰着脖子想看那最后一抹光辉，可那光辉迟迟没有降落，。心里那种似有若无的遗憾美很快就随着时间的流逝而消隐了。他叹了口气，懊恼道：“滚吧。” 黑夜终于得以来临。

这种扭转乾坤的神通手段我并没有亲眼见到过，是他在开公司会议的时候说起的。不过我确实记得曾经有一天晚上八点才天黑，我奶奶一直站在阳台上不肯做晚饭，说太阳神生气了，天要大旱之类的话。我曾经在公司会议作为代表发言的时候提出过要他表演太阳升降术，但是被他粗暴地的拒绝了。他问，你们想看么。我们异口同声地说，想。他敲了敲桌子，指了指身后的企业文化标语——“梦想破灭的地方”，说那就不让你们看。你们是粉碎别人梦想的人，首先必须以身作则，粉碎自己的梦想。

虽然他不肯表演太阳升降术，但是在别的方面对我们还算坦诚相待。比如他在每次有新人加入公司而召开的会议中都会向我们阐述一遍他为什么要开这样一个公司去粉碎别人的梦想。他说，他是一个很无聊的人。他的愿望全部都可以实现，所以他从来就没有梦想可言。但是，在这个世界上，总是有很多有梦想的人，非常的讨厌，循着自己的梦想而努力奋斗，过着丰富有趣的生活。他们实在太可恨了，走路带着笑，睡觉的时候可以做梦，还从未感到无聊。老板他承认，自己是一个小肚鸡肠的人，眼里容不下这些比他更幸福的人。所以，他要把他们的梦想去掉，让他们变得和自己一样。“整个世界充满了无所事事的人，啊，这听起来多么美妙。”

或许这也算是老板的一个梦想罢。我曾经疑惑为什么他不用自己的神通实现，而偏要征用我们。老板解释说，世界上有梦想的人太多，他管不过来。让我们来干这件事，我们干的慢，他的愿望就可以持续久一些。而且，自己可以在一边看，看着大家重要的东西咔嚓碎掉，这才能体验到快感嘛。

**二**

梦想粉碎公司分工明确，秩序森严，规模也十分庞大，分公司遍布世界的各个角落——这一点显而易见，因为哪儿都有需要粉碎的梦想。根据梦想的不同，粉碎的手段和难易程度也不一样。我刚入行时，被分配到了儿童部，负责粉碎儿童的梦想。

虽说处理对象是孩子，但这项工作其实并不容易，因为孩子们的梦想太过于丰富多彩了。要孩子再也不相信有圣诞老人很简单，只用在他爸爸半夜里换圣诞老人装扮的时候让他不小心看见就行了；可是要一个孩子不相信月亮上有嫦娥兔子还有砍树大叔这可怎么办。把他带到月亮上去吗。我还没去过呢。只好放一本《十万个为什么》在他的床头，打开《月亮是一个无人的坑坑洼洼的破地方》这一章，还得逼他看。

事实证明我的工作并不受到老板的重视。第一孩子的梦想不重要。第二，孩子的梦想随着成长也会自然而然的破灭。或许入职的初期总是要经历这样的岁月，干着最麻烦最讨人嫌的苦工，最后还被证明工作没有什么价值。而好处全被高头的人占据了。

在儿童部干了几个月以后我被调到爱情部做了短暂的停留，除了给出差的人买钻石跑车楼房什么的报销也没什么别的事情。后来因为这个部门的业务不景气，闲杂人等太多，我便又被调到了事业部当差。

在事业部我度过了自己职业生涯的黄金时段，因为踏实肯干思维活跃，我业绩斐然，最终成功晋升到了部门领导的位置。在我到来之前，事业部采取的粉碎手段主要以打击为主。动不动就制造家庭矛盾，天灾人祸，企图摧毁梦想者的意志。但是这样的粉碎手段有的时候会遇到问题。比方事业部曾经把一个做音乐的人双耳弄失聪，而这人最终还是写出了无数动人乐章；事业部还曾经把一个史学家小鸡鸡割了，但是他还是写出了宏篇史学巨著。这些失败的例子说明，打击作为一种粉碎手段有着致命的缺陷。所谓“杀不死你的会让你更强大。”，有些人的意志品质过于卓越，使得这种打击不但没有消灭他的梦想，反而使得他的梦想更加坚定。

在我接管事业部之后，进行了一次彻底的工作模式改革。从此以后事业部的粉碎方式由两条路线同时开展——一方面，事业部有选择的培养一小撮梦想坚定的人，对他们加以扶持，使得他们实现梦想获得巨大的成功；另一方面，事业部对其他抱有同样梦想的人进行普遍的打击。很快，同一梦想的人就出现了两派阵营，一派取得了巨大成功而另一派则普遍失败。这样的梦想粉碎模式取得了显著的效果。社会出现了不可调和的矛盾，失败一方的梦想持有者溃不成军，而成功一方也很快在胜利的喜悦里变得无所事事，庸俗无聊。这次改革的成功超乎我们的想象，并且这种成功迅速蔓延到了其他部门。社会矛盾的加剧让怀疑的情绪波及到了人们生活各个方面。悬殊的生存地位让很多人垂头丧气的接受了低贱的命运，嘲笑空洞的理想，并且怀疑曾经笃信的人生信条。很快，爱情部，友情部，真理部，道德部捷报频传。

在事业部的成功很快让我受到了老板的重视，我在不久以后升到了决策组，开始处理全公司最高级的业务，即如何毁灭世界性梦想。

所谓世界性梦想，就是那些被世界人民广泛怀有的，远大的光辉的梦想。那些梦想一般不会实现，所以就没有所谓成功者与失败者。正因为此，大家团结一心。这样的梦想很难彻底的消灭。只能用抓典型的方式，要是能让那些梦想领袖的梦想破灭，或多或少能够遏制这梦想的生长。以前决策组认为，那些人的梦想太难以破灭了。要是那些梦想杀不死，就把那些人杀死好了。那时我还不在决策组，我记得的案例不多，好像有一个戴圆框眼镜的歌手，有个微胖的美国黑人，有个古巴人，还有些中国人。不过好像决策组每决定杀掉一个这样的人，都会让更多人燃起更强的梦想，就像火上浇油一般。于是现在决策组不再轻举妄动了，决定让这些人在梦想永远不能实现的痛苦里慢慢的老去，那种掷向太空而没有回声的壮举最后肯定会默默的消亡的。我深以为然。于是我每天坐在决策组的办公室里，和我的同事们一起，抠着指甲，等着时光默默流逝。



**三**

老板现在很忙，每天要到世界各地检查梦想粉碎的情况。当遇到我们解决不了的问题的时候，他就会亲自出手，用自己的神通把问题解决。所以，在那些梦想重灾区，总是有很多奇怪的事情发生。要是你感到有点诧异，你就往天上看，看到星星划过，嚓，一个梦想破灭了。

我们公司也在不断的招募新人。人事部我没有去过，不知道具体的工作流程，但是我知道，大部分的新人都是我们公司业务开展的对象。在成功的消灭他们的梦想以后，他们会变得废柴，变得无聊，这个时候我们公司是最好的去处。我们所有的员工都不愁吃穿，生活安稳，只不过没有梦想而已。没有梦想，那有什么嘛。

只是当我问起那些匆匆走过的新人，你们之前的梦想是什么的时候，他们都表示茫然困惑，好像那种东西从来都没有过似的。我并没有怀疑他们撒谎，因为我自己也想不起来了。我之前的梦想是什么呢？在我来这间公司以前。好像那是一件跟我毫无关系，也毫不重要的事情。我想了想，想不起来，便任它过去了。



[![](60f0e1d8-b349-4c41-a48b-e92ac1bd8773.jpg)](60f0e1d8-b349-4c41-a48b-e92ac1bd8773.jpg)



我在粉碎别人的梦想的时候，会有一些愧疚。但是我从来没有觉得老板是一个坏人。相反，我觉得他博爱，仁慈，宽厚，只是有点无聊，并且感到寂寞。他想让我们陪伴他。而没有他，没有他创造的这个公司，我也终将是一个寂寞的人，或许也不会有什么梦想最后剩下。我现在觉得安稳，快乐，我看向我的未来，是一片空空荡荡的浅蓝。

我要说的我的故事也差不多说完了。明天我就要退休，回到家去，抠着指甲，任凭时间默默流逝。公司的新员工太多了，已经占到世界人口的一半。在这样多的员工的努力下，我们公司的事业也将蒸蒸日上。老板决定，让一部分员工离开公司，回到他们原来的生活里，做回他们的职工，演员，厨师，父亲，兄长和伙伴。这样，渗透了生活的各个角落，有助于他们更好地的完成自己的业务，彻底地粉碎掉人们的梦想。

走在街上，阳光强烈和风温柔。我看到人们，倍感亲切。喏，小张，哎，孙姨，那么多熟悉的面孔。你们好啊，亲爱的同事们。祝你们工作愉快。
