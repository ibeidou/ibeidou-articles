# <“寒假社会调查征集”系列特稿>支教随笔

**“诚然，我们无法苛求一种绝对意义上的教育公平，使得每个山里的孩子，都能享受得到城市孩子般的教育条件。国家近些年来义务教育免费化的政策，也不过是为了保证一种最基本的教育平等——普及义务教育。可是，我们仍要发问，制约了山里孩子接受高中以上教育的，难道只是因为单纯的个人贫困？”**

# 支教随笔

##  文/中国政法大学学生

09年寒假，我得幸与政法的其它15位同学一起，赴贵州仁怀支教。在喜头镇的卫星小学，度过了苦乐参半的11天。期间所见所闻，很多出乎意料。一些未曾想过的问题，愿与大家分享。 **一、酒都仁怀** 我们这次所到的仁怀（县级市）是一座山城，号称“中国酒都”，著名的茅台酒厂即坐落于此。据一位当地人讲，全县每年GDP中的70％都来自于茅台酒厂（此话的真实性待考）。其它大大小小的酒厂加起来，有四十多个。酿酒业构成了当地经济的支柱。 其直接后果有二：一是生态环境较好——由于当地几乎没有什么工业，也就没有污染（我疑心为了保障酿酒所需之水质、高粱等原材料的质量，所以一定程度上对某些污染性工业进行了限制。待考）。仁怀政府近些年的口号是“生态立市”，欲以此来发展旅游业。不过暂时看来，还没有什么效果。二是贫富差距较大——酿酒业的一枝独秀与其它行业（尤其农业）的发展滞后构成强烈反差。先富起来的一些人显得格外着扎眼。据一位在市政府工作的人大毕业的师兄所言，当地治安非常不好，警察上街巡视，都要荷枪实弹才敢出动。而在税收问题上，政府与酒厂间也存在着矛盾，相互间不是非常信任。 所以，虽然仅就GDP而言，仁怀去年已经突破了百亿大关，在整个贵州省县级经济体中名列首位。但对于很多平民而言，贫困仍是他们身后挥之不去的一个影子。我们这次所去的卫星村，正是这样的一个缩影。 **二、卫星小学** 卫星村隶属喜头镇（仁怀最穷的一个镇），全村3200余人，分住在几个山头上。村子的地理范围很大，为方便管理，村民们就居住地被分为若干个组。每组都有一个总负责的组长，还有一个妇育主任（负责计划生育的，大概是这样一个称号，记不太清了）。村里通讯不很方便，大部分村民家中都没有电话。加上住的分散，很多事情通知不便。一般的事项，都是由村委会去通知各组长，然后组长走访附近各家说明情况。但是近三分之一的组长也没有电话，于是问题显得更加麻烦。 我们支教的卫星小学，离村委会很近，大约位于全村的中心。但由于村子的范围太大，住的远的孩子，有的也要步行一个多小时来上学。若是往返，便得两个多小时。独特的地理环境造就了独特的作息习惯——没有午休。冬天，当地一般十点钟才上课（夏天是九点），一天连上五节(两节课后有个20分钟的大课间)，不到三点钟就放学了。孩子们回家后，一般三四点钟吃过午饭，然后下地干活或者在家做些家务。晚上八点左右，晚饭开始。 单就硬件而言，卫星小学的条件尚没有差得可怜。一座三层的小楼，每层一小两大三间屋子。大的做教室，小的做办公室，六个年级正好放下。只是一年级要和学前班共用一间教室，年龄跨度较大，不好展开教学。万幸小楼旁边的一排平房（共四间）今年终于整修完毕（成为危房已数年），下学期可以投入使用。学前班的问题宣告解决。一个操场略大于篮球场的面积，三百多名学生，体育课不好开展。但对于完成最基本的教学任务而言，这样的条件，足矣。 全校老师十余名，每个年级两三人。主要是语文和数学（高年级也有英语）。课表上的课程很丰富，社会自然品德等等俱全。但大都没有老师来上。和我们小时候语文数学老师挤占“副科”上课时间不同，这里的老师并没有太多的上课热情。但凡所谓副科，一律安排学生自习。而就是这样一群缺乏热情的老师，使得孩子们的学习状况陷入了极端糟糕的境地。 **三、家庭教育的无力，学校教育的失职** 山里的孩子一般家境都差，据我们十来天家访（十来个人共走访了近百户）所了解到的情况，大多仅仅满足温饱。半数以上的家庭都会有人外出打工，这样每年会有数千元收入。若是全家务农的，则除去日用花销，一般所剩无几。 山里交通不便，物资相对匮乏，田间家里，要做的工作相应多了不少。通常情况下，家长很难有闲来监督孩子的学习。至于辅导，则更没有能力（通常文化水平有限）。这一情况，在那些外出打工的人家尤甚：一般外出打工的都是家里的主要劳力，剩下的老弱打理田地已是相当艰难，更无闲暇关注孩子学习。何况很多孩子也要负担不少的家务。山里的孩子一般又“野”，跑出去了家里也难管（尤其是那些父母在外，爷爷奶奶代为照看的）。加上家长大多看不懂孩子的作业，无力照看学习。便把对孩子学习监督的希望，便都寄托到了老师的身上。 然而不幸的是，这里的老师大都失职。 由于酒业的推动，仁怀经济总量很高。于是相比起百姓，政府显得相当有钱。政府财政拨款的公务员、事业单位工作人员的待遇，也就都很可观。所以，即便是最基层的乡村小学，每位老师的月收入也会达到一千七八，远远超过了当地农民的生活水平。（就我个人的家访经历，家庭条件明显好于其它村民的人家有两户：一家孩子的大哥是老师，另一家孩子的父亲是赤脚医生）所以，对于很多老师而言，在卫星村的教书生活几近“流放”，与自己的收入地位并不相符。大部分人并没有什么热情来做好这份工作，反而一定程度上鄙视学生与家长，对自己的处境多有抱怨。 据五年级一位名叫胡小美的女孩对我们讲，一些老师打骂学生相当严重——“他瞧不起我们卫星村人，嫌我们穷。还说我们卫星村人就该穷。” 支教期间，我们曾要求各个孩子写过一张条子，就我们的教学形式与内容进行反馈。有一张上是这样写的：“这几天，我很高兴，因为有北京的老师给我们上课。他们都很好，不像我们这些老师一样，只知道打牌呀，麻将呀，我恨死了！……希望所有的老师不要走，请留步！”拒不完全统计，接近半数的孩子对我们提出过挽留的要求，或者希望我们能够给他们换来新的老师，因为他们的老师都不关心他们，不批改作业，不解答问题。“上课听不懂的，下课问老师都不回答，只说让我们自己好好想想。作业老师只检查你有没有做，但不批改对错。”其结果便是，不懂的，便一直不懂，甚至不知道自己懂不懂。 据我们了解，这些老师大多是近些年新换的。五六年级初入学时，还是一些老教师在讲课，所以基础相对还好。四年级以下，几乎学得一塌糊涂。三年级的孩子，不少尚认不全汉语拼音，背不会九九乘法表。 三年级曾布置过一次作业，以“我的理想”为题写几句话。一个孩子这样写道：我的理想是当一名校长。因为**校长能管很多人，还有很多钱。** 而在我们支教的最后一天，我们终于见到了这位校长——在镇上的一个饭店为我们设宴送行。在此之前，他始终没有出现。校长家住在市中心，据喜头镇约一小时的车程，再从镇上上山入村，奔波起来也是十分辛苦。据校长所言，他毕业后即到卫星小学任教，三年后任校长，至今已经十一年，可谓把青春都奉献给了这儿。送行宴上，由教务处主任、财务以及两位女老师作陪。酒过三巡，校长开始微醉，语言轻佻，小动作调戏女老师，也让我们见识到了他活泼的一面。 **四、免费教育的后果** 老师们的不负责任，固然来自于其与家长们经济、地位的差距。而义务教育免费化的实施，也在一定程度上加剧了这一情况的出现。 由于义务教育开始免费化，小学毕业生全部可以进入初中就读。于是升学率的概念便从小学彻底消失。“教好教坏一个样，何苦让自己受累呢？”卫星小学的老师们，大都是抱着一个这样的心态。 而另一个令人始料不及的后果便是：**免费的教育，也在一定程度上降低了农村家长对孩子学习的重视！**据卫星村一位村民介绍，以前上学要钱时，很多家庭都会感到压力巨大——每逢开学，都要四处借钱，来凑足这几百元的学费（一般人家都不只一个孩子）。正是由于这钱来之不易，家长对孩子的学习特别在意，不努力时甚至棍棒相加。而现在免费了，一年只需十五元，很多家长便不再那么重视，全凭孩子自己的自觉。初中毕业时，若能考上高中，再考虑供或不供。若成绩不好，则单纯地认为“不是上学的材料”，心安理得地放孩子外出打工。然而由于当地教学条件所限，加上孩子的自制力一般都较弱，很少有孩子能够考上高中的。 **五、我们的教学生活** 因着当地特殊的作息习惯，我们因地适宜地调整了自身的教学生活时间表，情况大体如下： 早晨7点钟集合（一般六点半左右起），在村委会前的广场上练会儿操（后来的几天是排练离开时联欢会的开场舞）。八点钟吃过早饭后休息片刻，九点钟向学校进发。到校后，班主任（十二名志愿者分到六个年级，担任这段时间的临时班主任）带领各班早到的学生打扫下教室，稍备下课，十点钟课程正式开始。 与学校日常上课不同的，我们在五节课后加了一节自习，方便班主任和班级学生的沟通，同时也解答下课程问题。每天课后，各班主任会联系好本班的二到三名同学，随同他们一起回家家访，了解家庭情况，并鼓励家长督促其学习。 为避开村民们吃午饭的时间——山里人好客，必留人吃饭。我们常不放学后马上家访，而是先回驻地吃饭，约好学生饭后来村委会找老师。无奈一些学生离校太远，往返山路要两个小时。这时只能不吃饭随他一起回家，家长问起撒个小谎但言吃过，回驻地后再补。 一般而言，每日家访结束后回到驻地的时间都在五点半到六点间。六点半时若尚未回来，一定要和领队联系——届时天已将黑，山路不安全。七点钟开会，讨论一天上课的情况与家访时获得的信息，还有一些日常的事务。一般都要一个半小时左右。九点左右晚饭开始。然后自由活动。十一点后，大家陆续进入梦乡。 令人欣慰的是，每日的例会都有内容，并非流于形式。尤其是前几日，几乎每个年级，无论教学还是家访的过程，大家都能发现新的问题。各班主任依次阐述自己遇见的问题和个人观点，然后大家各抒己见进行探讨。家庭教育问题、义务教育免费化问题、支教目的问题，都是在例会上提出的。待到五六天后，教学步入正轨，当地的情况也都基本了解，例会方才精简。 我们也曾反思，来这里的十来天到底能够带来些什么。当那些孩子不停地问我们能否留下或者为他们换来新的老师；当我们看到他们的基础如此之差需要补的东西如此之多；尤其是临别前的一日，当我们发现很多孩子并没有太多的感伤依旧更关心下课后出去玩耍——我们怀疑，自己这些日的努力，到底有着多少的效果，在他们的记忆里，又能保持多久。 但是，仍有一个理由能够支撑着我们继续努力，那便是，一旦这影响能够形成，它的效果将会惠及一生。很多人都曾有过这样的经历，一个好的老师，改变了他们对学习的看法，或者一个人的信任或爱，让他们对世界重新认识。除了具体的知识，我们也带来了一些理念，给他们带来了原来老师不曾给予的关心。虽不知这些东西，能够影响几个孩子。但我想，对那些真正珍惜体会的孩子而言，这获益将是巨大的。就算只是为了这几个孩子，我们这些日子的努力，也没有白费。所以，虽然对当地情况的了解在前几日时便已完成，接下来的每天，我们仍然坚持家访。只愿通过这种方式，能够让孩子和家长感觉的到关心与重视，通过对家长的鼓励（有时这甚至比对孩子的更重要！），让他们明白，他们的孩子，也可能拥有一个美好的将来。为此，需要家庭更多的监督和鼓励。 **六、张力：整体经济水平与村民生活之间** 仁怀触目惊心的贫富差距，让我们切身感受到这样的一个事实：国（当我们已经太习惯于将其等同于“政府”，这里不妨也类比适用于一处之地方政府）富可以同样民贫。其政府大楼之恢宏、市政广场之开阔，早已不输任何城市。茅台酒享誉海外，自厂长至员工也都盆满钵满，而穷困如卫星村村民者，又是如此普遍。 据那位校长讲，茅台酒厂的待遇相当之好。一般进厂七八年的员工，每月工资至少在四千五以上。各种福利又是样样俱全，大件的酒油米，小件的肥皂糖，皆有发放。而且假期还长，一般每月一半时间都在轮休。一家人有一个在酒厂工作的，家里的用度就都不愁了。所以，进酒厂也就十分困难。关系、文凭（大专以上）、钱（打点诸方领导，约十万元），缺一不可。而仁怀的关系网又很复杂，于是一般人只能止步。 所以，因酒厂而获益的，仅是当地一小部分**自身条件本就不错的人和政府（高税收）而已。至于当地普通百姓，并未因此而获得更多的就业机会，他们享受到的，只是那被富人们抬高的物价——**据一位学生家长介绍，当地读高中，一学期的学费是一千二到一千八之间。而生活费尤贵，最节省的孩子每日也要12元才够用（这在一般的县城极其难见）。读一年高中，每个孩子都要数千元。这对于大部分农村家庭来说，几乎难以承受，更何况每家一般都不止一个孩子。 至于政府的各项补助，也是相当有限。据一位村民介绍，大多是发给那些有残疾的村民，每年数百元。如她家这种，每年收支刚好平衡的，便拿不到补助。很多村民都面临着这样一个尴尬的局面：若说贫困，尚能满足温饱；至于发展，包括让孩子有机会接受高中以上的教育，则是困难重重，几无可能。 一个很有趣的问题，便是计划生育。和笔者家乡农村的情况不同，这里在农村也落实“只生一个”的严格政策，于是家家户户几乎都要被罚款。而罚款金额，则从一千到七千不等——视与主管官员的关系而定，活动能力强的，上下打点，便可少罚甚至不罚。相比起每户的年收入，这一金额的罚款相当致命。 **七、反思：分配公平的最终制约** 笔者对于经济学所知甚少，对于如何带领村民致富发展的问题并无答案。然而若以卫星村孩子接受教育的情况作为分析的起点，一种对**机会平等的诉求**仍然跃然纸上。 诚然，我们无法苛求一种绝对意义上的教育公平，使得每个山里的孩子，都能享受得到城市孩子般的教育条件。国家近些年来义务教育免费化的政策，也不过是为了保证一种最基本的教育平等——普及义务教育。 可是，我们仍要发问，制约了山里孩子接受高中以上教育的，难道只是因为单纯的个人贫困？ 酒业的发达，一定程度上限制了其它行业的发展（例如物价上涨的因素）。政府从整体经济效益的方面考虑，扶持了更有效益的酒业，也在一定程度上，令其它人来承担了这一政策选择的成本。理应通过对税收的二次分配，来对负担成本的人民进行补偿。 然而我们在当地看到的却是，**为实现分配公平而存在的义务教育也成了利益分配的战利品**——如我前面提及，教师的待遇在当地也算可观。笔者疑心这一职位分配的背后，是否也会有着其它因素作祟？更具体点儿，我怀疑这些老师和茅台酒厂的工作人员一样，不少也是走关系进来的。（在学校的储藏室里笔者无意中发现了老师们的档案，大部分都是函授的文凭。与一些老师也见面，男老师居多，一般20来岁，不少都带些痞气。于是不难理解他们为何没有教学热情——本就不是为了教学而当老师）这使得义务教育的普及只能在形式上得到成立。就实际情况而言，这座教室完备老师不缺的小学内，学生真正接受的教育依然有限，直到小学毕业，一般也不过三年级水准。这使得的他们在接下来的求学生涯中极为被动，很难获得升入高中的机会。 加上计划生育的区别罚款，以及物价因素影响下带来的高额高中学费，我们不难发现：**除去个人因素所导致的贫困，政府在参与二次分配过程中的不公平行为，更使得平等的教育机会在当地变得更加困难。** 在某些情况下，相比起经济总量的增长，分配公平才是制约某些贫困人群改善发展自身现状的重要因素。 （注：由于笔者此次支教的身份不是记者而是老师，所以大部分精力都放在了教学和家访上。此文只是随笔，多是些道听途说的信息，未经考证。对当地情况的介绍若有不实之处，望大家批评指正。）
