# 夏夜离人殇

## 文 / 楚陌（山西大学传播学研究生）

 

[![tianxuan](/legacy/2009/06/tianxuan.jpg)](/legacy/2009/06/tianxuan.jpg)

 夏夜照例的书法班之前，随意拐进楼下便利店拿了瓶果汁。冰柜里面有最显眼的那瓶，通体红色的魅惑包装——以前没有见过这种果汁饮料，半个石榴的图案诱人的闪耀，喝一口，冰凉的酸甜。 忽然想起我已经仙逝的姥姥。 那时我们家在山上自己建的房子，有不小的院子，院中苹果树亭亭如盖，西院邻居的院子里树种更多，其中便有数株石榴。 忘了是邻居相赠抑或妈妈买来，总之记忆中初中的夏日，回家后经常会看到一粒一粒剥好的石榴，红宝石一样的堆砌在水果盆里，姥姥喜欢笑眯眯的看我吃完，然后笑着问嘴角沾着红红果汁的我：“再给你剥一些？” 可惜当时我尚年幼无知，不懂得珍惜这莫大的幸福，亦不懂得，在失去老伴之后，姥姥是怎样一个人咀嚼她的悲苦，然后转身又对着我们慈祥的笑。姥姥有心脏病，初中时候有很长时间住在我家里。至今想来，仍不知不能多做运动的姥姥每日在我们上学，爸妈上班之后，是如何独守静谧而荒凉的时光。 姥爷是在我初一时候就去世的。幼时我性子甚野，妈妈讲出生时候护士拍打我都不哭，在姥爷下葬时候，我们几个孙辈小孩甚至觉得好玩，跪在各自的父母背后挤眉弄眼装作嚎啕一阵便好奇的在灵堂大人们的胳膊弯下面钻来钻去，惊讶于放哀乐的音箱声音可以如此巨大。而我们的家长，在灵堂那边抢地悲泣，嗓音嘶哑欲绝。 现在想来，当时的年幼，简直可以用没心没肺来形容。 但那些美好的小事，我全都记得。我记得上学前姥姥姥爷装作无意的掏出零钱笑着与我，我记得姥爷在我放学前早早的在桥头等我，我记得姥姥姥爷在逗我给他们脱完袜子骄傲的对邻里夸耀我的孝顺，我记得清早起来姥姥姥爷忙前忙后为我张罗早饭——我从小特别挑食，喜欢吃胡萝卜，但只喜欢生的不喜欢熟的，喜欢吃方便面但是怕咸，于是当时的早餐便是姥爷细心地煮好面，斟酌着放好调料包，姥姥也很利索的切好一小碗萝卜丝，拌上山西的陈醋，在熹微的晨光中，看着我吃完早饭，抹抹嘴就去上学。叮咛我好好学习，路上小心。 我漫不经心的答应着，昂起脸迎着太阳去学校，直到现在，回头望时才惊觉——那时他们应该都是没吃早饭就起来为我张罗的。 亲情确实是很奇妙的东西。有无之间，能让感动与幸福充盈天地。 姥爷去世之后姥姥精神一下子大不如前，在我第一次高考前几日，便随着姥爷而去。姥姥姥爷在孙辈之中跟我在一起的时间最长，也自小最宠爱乖巧又顽皮的我，可惜——当时我还不懂爱，不懂这样日复一日平淡的祖孙生活，这弥漫着世俗气息的市井生活，之于漫长又短暂的人生，是何等奢侈。 我是在某年某月一夜长大的，那之后心性性格均大变于昨，而从那夜直到现在，我依然有时常陷入回忆中的习惯。以前常常会对自己说：如果能有一次机会，我会毫不犹豫的毁灭整个世界，再换回那一段我懵懂中的幸福时光。 可惜幻想只能是幻想，斯人已逝，心里面柔软的一部分也在那个夏天丢在那段记忆中。“殇”是一个不动声色的字眼。静如止水，在不为人见的光阴中传递着穿越时空的哀怨，以沉痛的姿态，为转瞬即逝的生命赋予一个欲哭无泪的角度。 姥姥姥爷终于没能看到我以后的路，犯过多多少少让人警醒的错，做出大大小小有趣的事情，交了南南北北有趣的朋友。少时会自囿于一个被师长圈定的道德规范的范畴中，会有人来告诉你，什么是好的，什么是对的。现在知道了，师长们口中的君子之道，他们自己其实也不甚明了——孔子在回答什么是君子的问题时，只有一个简单的定义：不忧不惧。而“爱人”是仁者之道，“弘毅天下”是为士之道。想做个君子，行事无所挂碍，进退随心便已足够。（司马牛问君子，子曰：君子不忧不惧。曰：不忧不惧，斯谓君子已乎？子曰：内省无疚，夫何忧何惧。）而我于君子，亦有异见。每个生命都是绵长的大河，有人喜欢静水流深，完美的饰演一个优秀的社会角色，我欣赏这样的人，而我更愿意在同样的生命长度中体验多一点的生命角色。而形形色色或喜或悲的人生体验，都让这个时段的生命变得丰厚起来。旅途中两旁吹来的赞叹或批判的风浪，都不会影响到我心里的航向。 各派书法我最喜小篆，而每次练字之时一笔一划都像将自己的性情投入一般。圆融随性，挥洒自如的线条，其实是对于自己人生最终境界的希冀。每每一笔一划写出，都像读到一本好书般欣喜，我于方寸横竖间清楚看到十年，二十年后的我，放下笔，又回到略带不羁的二十岁。 这样就很好。那样也很好。我能做我便很好。 佛学中关于“我”与“佛”的公案浩如烟海，而“我”又是每个哲学流派首先面对和要终极解决的的问题。答案是什么，有多少种，一点都不重要。每个个体的人都有不同的智慧层次，人生阅历，“我”是面魔镜，你将怎样的自己投射进去，便会得出怎样的“我”的答案。而觉悟二字，学字头，见为底，吾心为悟——很清楚的讲明了这是一个学习见我心的过程。 是夜，一场小雨路过，推开着窗望着渐沉得夜色，希望我爱的人，爱我的人，不管在人间还是在天堂，不论见证了属于自己“我”的心性还是暂时自性蒙尘，愿你们都安好。水流经城市变得深沉，我经过人群改变许多。感谢你们在自觉或不觉中陪伴我一路蜕变。 《金刚经》云：通达无我法者，如来说名是真菩萨。  (编者注：经作者授权，全文略有改动。)
