# 沙民系列之二﹒夺命刀

**他每动刀斧之前，必然吸食大烟，叫病患家属或助手用麻绳将病人绑在条凳上，严严实实无法挣动，而后将白刃从烧酒中取出，用白布缠柄，一面抽烟，一面把刀放在炉火上烤，待到刃口微微发红，蒙上患者双眼，深吸一口大烟，双目陡然增光，灌注全身力道于刀口，从患处疾刺而后拔，一气呵成，伤口必有脓血流出，待血污流尽，用湿布擦净，敷上草药。据说刀速之迅猛，往往刃已离肉，病人尚不能反应，过后才呼喊嚎啕，偏偏刀法精准出神，从不伤筋骨动脉，旁人看过，叹服不已。**  

# 夺命刀



## 文/周阿飞（南京师范大学）

 

生我那年父亲还未入西厂，他高大清瘦，面孔如刀劈斧削，青茬似钢针倒立，帅气刚毅，挺拔得像东疆岸上的水杉。做学生时当过六年班长，处事一马当先，待人三分义气，更兼练得一手好字和一副清爽笑容，前二十年，哪家的姑娘提起曹圩周家的二郎不是眼睛发亮心发酸。可惜他最终也没有考上大学，两届都不取，读了两年农干校之后，就回来跟着师傅学做烧饼。再后来就相中了我母亲，结了婚。

同年冬月初五夜，人家都熄灯了，母亲肚子开始疼，父亲骑着结婚时陪嫁的凤凰黑皮大杠车，驮她先走，奶奶起身穿了衣服，用布围子包好先前做好的尿布和小衣裳，朝前头追。那一年的雪压断了晒场谷舍的黄杨木横梁，那晚下了整年最大的雪，奶奶赶到医院时已接近子时，风雪打得她睁不开眼睛，医生还没来。没法子，扔下布件，甩腿去请。

医师是上了年纪之后从北边儿退回来的，本不在卫生院挂号，可院里唯一的妇产科医生回家探亲去了，他又常给户上的牛羊牲口接生，只好将他请来，等老人家赶过来，我已经连头带尾的出来了，老先生捏着剪子上来，朝着脐带，准是一刀快的。抓住脚把这丑东西倒提起来，屁股上一拍，哇哇地嚎起来了。温吞水擦干净身子，老棉絮一包，往磅秤上过一场，八斤八两。

第二天上午，父亲穿着破了洞的咔叽布裤子，拎着红蛋去外婆家报了喜。

这医师确有本事，而知道他且知道他在沙上的人并不多，却正好有我家老大人。爷爷曾经跟我讲过一桩，四清五反时，医师成分不好遭批斗，却因为手里的回春之术医好了某位首长的顽疾，是而无论受到何种斗争，总每天保证他一只老母鸡的营养。后来说是闹革命，首长也被闹了，要他交代这个大坏分子的“反革命证据”，他“不大配合”，就让“滚出来了”。

他属中医，却不避手术，尤擅以快刀治骨肉感染，我长到七八岁略通人事的时候，每读三国都怀疑其为华佗传人，只是不见他用麻沸散，是以不能确定。不过爷爷曾亲眼见其动手术医人，说他每动刀斧之前，必然吸食大烟，叫病患家属或助手用麻绳将病人绑在条凳上，严严实实无法挣动，而后将白刃从烧酒中取出，用白布缠柄，一面抽烟，一面把刀放在炉火上烤，待到刃口微微发红，蒙上患者双眼，深吸一口大烟，双目陡然增光，灌注全身力道于刀口，从患处疾刺而后拔，一气呵成，伤口必有脓血流出，待血污流尽，用湿布擦净，敷上草药。据说刀速之迅猛，往往刃已离肉，病人尚不能反应，过后才呼喊嚎啕，偏偏刀法精准出神，从不伤筋骨动脉，旁人看过，叹服不已。这时再看医师，颇为萎靡，因为妙法难施，每出一刀，必然集中全副精神，消耗甚大，故而每次刀诊都要吃用大烟，积聚气力。

到此方能明白，华神医的麻沸散是给别人用的，而且他儿子正死于药的主方，可见是药三分毒，医师给自己用，不给患者增添负担，我看这上面医师是超过前辈了。可惜等我出生时他已不动刀多年，下面的徒弟多是走的现代化的路子，就算有问起的，医师也只是笑而不传，这手艺也就断了。许是如同老舍笔下的神枪沙子龙？哪个晓得。到是我常常想到他那些大烟从哪里来的，也是和老母鸡一样特供？哪个晓得。

老医师于我的大恩，是在我过到四岁多的时候。正是床上桌下满地爬的光景，别家的小子活蹦乱跳，我却生了一场怪病，脚沾不得地，这病现在叫阴囊坠胀，俗称大卵子，双腿一发力就病发，疼痛异常，如同被持续阉割一般。按说男子出生之后大都有此经历，唯独我小儿时久病不治，因不能站立，都是奶奶用麻绳将我缚在背后，一刻不离。其时正毁旧家盖新楼，工程不小，奶奶为我牵累不能帮工，祖孙二人皆深受其苦。遍寻医方而不得痊愈，卫生院传信说要动刀，不然不得除根。家中顿时一片愁云惨淡。

我父担心睾丸动刀，一个偏差岂非阉割？我一听刀斧加身，岂不类于杀猪？想那猪被尖刀刺喉时何等狂呼，可见疼痛，一想起就急得哇哇嚎啕。正进退维谷之际，卫生院一个病友让我们去问问看门的老大爷。

看门的就是老医师。未及叙重逢之喜，先给脱裤子一看，医师说要不先试试土方子吧，不行再开刀，一家子如蒙大赦，连声感激。

我回家吃了半年荔枝干，喝了半年荔枝壳泡开水，再下地时，身轻体健，百米绝尘，只是流了好几次鼻血，荔枝性阳，三伏天里虚不受补。父亲架着我让我骑到头上，提上十斤干白果去谢他，医师看见我们，很高兴。

那以后我就常去找他，他嘴里的故事比爷爷还多，爷爷在村上做事不常理我，他不一样，他见我跟我见他一样乐意。他大号马继武，祖籍高邮，世代出医生的回民人家，通诗词，还谙音律，一笔瘦金极是俊秀。年轻时跟随一位游方行脚僧行医，算半师半友，去过许多地方，救过许多性命。

他住在堂堂圩街上垃圾站的旁边，我去一次要走好久，但他那里有特殊的魔力，叫我难舍难离。一是他的铁炮。我这个年龄往上的孩子该是大都见过，一个黑乎乎的椭圆铁壳罐子，里头灌上粟米和糖放炉火上烘烤，不多时一声暴响，黑烟腾腾，打开罐口，倒出来白花花甜黏黏的炒米糖。我每次去都眼巴巴的看着那铁炮，他一般不让我失望，放下手里的活计，生炉子架炮，“砰”！我们三个眉开眼笑。之所以是三个，因为他这里还住着一个小哑巴，说是他孙子，但我估计是老医师捡来的，因为虽然他俩眼睛底都透着蓝腚色，但是小哑巴的头毛黄得发亮，而且没听说老头有儿女啊。

他用一张牛皮纸裁成三份，我们仨各分得一捧，他扔一块糖进嘴里：“黄儿一份，阳阳一份，老头一份！”边吃边咂嘴，我也咂嘴，又香又糯，咂得“啧啧”响，黄儿也想咂嘴，可他咂不出来，就急得要哭，我不要他哭，就教他，他怎么也学不好，我也急了，就要他张嘴，一看他的舌头比我们少了半截，像是被剪掉了一样。我咂不出嘴了。他看我不咂，也不哭了。

黄儿随老头姓，大名马君瑶，也是老头起的。我一听这名字可真俊啊，不想让他叫，就喊他黄儿，时间一长老头也随我喊顺了口。我猜得不错，黄儿是被老头捡回来的，碰见他的时候他正扯着老头儿的衣服讨吃，瘦的只剩一层皮包骨，舌头被人剜掉了半片好出来讨钱。老头给他吃炒米糖，带着他转了好几条街甩掉后面吊着的人贩子，给带了回来。

后来一直到我上小学三年级，黄儿都没入学，不是不送他，就是不肯去。以前要饭的时候好容易得点东西都被大孩子抢走，还要打他，他怕。

我总教他学咂嘴，渐渐他也摸出点门道，使劲把半片舌头卷上去，抵住上腭，总算能发声了，不过我们都是“啧啧”，他是“咯咯”，一咂一声笑，一边拿炮上的灰把老头儿的眉毛染黑，老头子用锅灰在我们脸上写字，弄完了去照大镜子，哈哈大笑，摸成一团。

我入学以后去老头那里的时间要少一些，他有时候来小学门口卖炒米糖，带着黄儿。每次我举着班级的小牌牌走在队伍最前头，黄儿就眼巴巴的瞧我，有点儿羡慕，有点儿陌生。

冬天我会带点儿蚕豆和花生去爆了吃，有时爷爷也让我给他带点烧酒。他很欢喜，但好酒而不善饮，一杯下肚就给我们讲故事，讲打日本鬼子。似乎他那半个和尚师父曾经杀过鬼子兵，还组织过僧兵队，唤作“荡魔金刚”什么的，几天几夜不吃不喝，照样上阵杀敌。讲师父医术高超，悬壶济世，做大善事，圆寂之后肉身不腐，总之万分神奇。

“师父医人比你医得还好吗？”

“我哪里比得上师父，我的都是他教的。比不过，比不过。”

“那怎么没有听说过，他有华佗厉害吗？”

“怎么没有听说过？咳，华佗是古人，不好比。想当初，民国二十年两淮发大水的时候，谁不知道我师父……”

他一讲这些民国之类我不懂的，我就不喜欢，赶忙的发问：“发大水是什么样的，很厉害吗？”

“咳，厉害啊！告诉你吧，鱼都往人家家里面游，树上挂满了蛇，高邮湖的水把城墙都冲塌了！”

我们如痴如醉。黄儿也问，他用笔写下来，他的大字写得真漂亮，我怎么都学不会。这都是老头儿天天教的，我酸酸地想。可是他还会抓麻雀，捉蚂蚱，做纸长蛇，用树叶子吹叫叫，这也是老头子教的？

可惜故事总不能一直持续，他一喝醉，我们就没机会提问了，因为他会说胡话，要么就是取出一把旧二胡，咿咿呀呀的拉起来，酒气冲天：“扶灵牌不由我……”像是唱花鼓戏，但是我们不爱听戏，有时唱着唱着还哭起丧来，我们更加躲得远远地。若是现在再听到那沙哑的唱腔，应该能品出悲悯的味道吧。

他还喜欢给自己画花脸，画之前要我们给他捏胡子，下巴壳底下的他看不着。我让黄儿搬住他的头，我蹲在下面用镊子捏住一根，我的手猛地一收，他的脸就跳一下，然后把粘在镊子上的白胡子擦在他手上，匀成一排，数来数去，我们仨就都欢喜了。

他自己修眉毛，修的很仔细，左手倒握了柄，右手拇指与中指捏住刀尖，一点一点地划过去，姿势非常好看。我知道这是什么刀，他说过，叫压衣刀。我时常想，割人家皮肉的就是这把？他没有颜料，多是用墨汁白灰，但是很认真地涂抹，不留一分空白，春天还往头上摆一枝红花。要是逢到他开心，还会给我们画。画完了花脸就上街去收垃圾，边走边唱，有社戏，有号子，也有乡谣：“



红娘子，白艳艳，

颈头上，银项圈，

大红兜，绣鸳鸯，

红罩衫，滚花边，

青绿裤，花圈裙，

花绣鞋，俏刮刮，

………”



我们跟在后面满心欢喜，还跟着和，黄儿拍手我唱歌，一人也插一枝花。

有时他唱着唱着就流眼泪，就大喊大叫，就发疯打滚，样子很吓人。我不喜欢，跳得远远的，黄儿不肯，守着他。等到不疯了，我们把他扶回去。他一回家就会狠狠地咳嗽，身子弓得像虾，不是断断续续的，像电钻在墙上打眼，像洋钎铲在石头上，非常渗人，没有痰，半夜吐过血，黄儿告诉我的。我总觉得他发疯是不是犯了烟瘾，好在从不伤人。

奇怪的是，我去他那里从来没有看他抽过大烟，不过他也没动过刀子。只一次我奶奶不知是腰上哪块疼，十多天直不起来，就来找他，他取了一根三寸多长的银针，一针没下去三分之二，再搓着缓缓拔出，溅出一股血来，都发黑，腰背立时直了。

他自己的背倒老是直不起来，咳嗽的日子越来越多，还时时见红。我头一回感到担心是什么滋味。有一回他在太阳底下修脚，我终于也忍不住了。

“马嗲嗲，你多大啦？”

“你爸爸还没你大的时候我抱过他，你说我多大，哈哈。”

“那你会不会死啊？”那时我的英雄本家太爷刚走不久，我知道死就是躺下了再也醒不来，不能吃不能唱，任你多少好故事也张不得嘴来说。

“哈哈哈哈，那不能的，二十年前在河南，板桥的大水都淹不死我，咳，不要紧，咳嗽不是病，死不掉的，哈哈！”他抬头热热地看了我一眼，晃一下头，又低下去咳，脸色涨得通红。

这板桥什么的我又不知道了，但是他说不会死我听懂了，一下子释然了，也伸着头兴致勃勃地去看他用剪刀割鸡眼。

这人间最美妙就在于前路不可测，最可悲也正由此。没过多久，我就亲眼见到了一场震动九州的大水。

九八年夏天，没日没夜的倾盆大雨下了个把月，沿江一线全部告急，真个就鱼行堂屋，蛇盘枝头了。父辈的男劳力们全部上了大堤，我们困守家中，老大人们愁眉苦脸地望向圩田，我心底暗暗地兴奋，老头子说的是真的，这雨快些停吧，水早点退，我好去告诉他，这大水的情景真应了他的话。

等到水退之后，我去看他们，已是人去楼空。

我父亲问院长，老医师人呢，院长说，死了。不到两天就葬了，说是水太大怕被冲走，三丈二的白布裹着运回高邮葬在什么山上了。

死了。整日的下雨，火都打不着，洗礼的水干净吗？黄儿不能说话，不知谁给念的清真言？

院长说老头子是一把好刀啊，年轻时不知医好了多少外科感染、结核病，人家说他是和阎王争性命，起个名号叫“夺命刀”，病患很多都是带着大烟来看病。就是身后凄凉点儿，大革命时老婆跳了湖，有个女儿走了失，再没回来过。

他给我留了一把压衣刀。我当成心头掉下的肉一刻不离身，这定是当年他用过的那把夺命快刀吧，我时常想，不知割去了多少垢肉，迎接了几多生灵，刮掉了几许细眉呢？
