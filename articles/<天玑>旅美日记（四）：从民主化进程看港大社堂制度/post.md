# 旅美日记（四）：从民主化进程看港大社堂制度

**“所谓意识形态，就是把不服从的人都抹黑成人民公敌，指责他们阻碍了社堂文化的伟大复兴，颠覆了乌托邦的社堂理想。然后对这些人冷眼相待。在这样的环境中生活是很不舒服的。虽然没有什么具体伤害，但是精神的压力也是一种伤害。这是威权制度的惯用伎俩。香港的繁荣来源于对个人自由的保护，而个人的自由包括不受歧视和言语侮辱或精神压力。社堂文化的实践，和自由民主的理想背道而驰。”**

# 从民主化进程看港大社堂制度改革的势在必行

### 

## 许子辰（香港大学）

### 

首先有必要描述一下港大的社堂制度。港大的社堂是一种由学校监督，但是运营相对独立的学生宿舍。社堂的最终权力来源是舍监，理论上舍监可以在法律规定的范围内做自己想做的任何事。每个社堂都有自己的社堂文化。有一个选举出来的学生组织去维持，发扬社堂文化。这个学生组织有负责组织面试候选堂友的权力，以及向舍监建议哪些堂友应该被从群体中剔除的权力。但是这个制度有很大的缺憾。以下我将从公共行政开支以及民主实践两个角度论证港大社堂制度改革的势在必行。 首先，从公共行政角度看，港大社堂的宿位享受着巨额补贴。港大是一所公立学校，对宿舍的补贴不管是直接来源于政府，还是来源于学校，都可以归结为公共行政开支（捐助另当别论）。在西环下租房，每个月租金至少两千港币。薄扶林道作为富人区其租金理应更贵。然而港大宿舍大部分的月租金是一千港币，而且还有公共活动室（pantry）的资源可以自由使用。这其中的差额，就要靠政府补贴，可以想象其数目之大。而政府补贴的目的，在于使住房有困难的学生能够有个栖身之地。例如大陆生，在香港本无立锥之地，在住宿上需要帮助。再例如住在新界的香港本地学生，每人要乘公交地铁一两个小时来学校上课，其时间金钱成本都很大，如果有住房补贴让他们在学校周围就近住下，将是一大功德。这也是政府补贴的初衷。但是当前的社堂学生会（student association，简称SA），刻意混淆了两个概念。他们把对住宿的补贴说成是对社堂文化的补贴，这就大错特错了。 社堂，在美国其实也有类似的组织，叫做兄弟会（fraternity）。要想加入兄弟会，就得按照老会员的要求去做一系列傻事，比如凌晨在海滩上吹五个小时的海风，或者赤身裸体戴个面具跑到某个正在上课的教室去闹一闹。经过一些列考核，才能成为兄弟会成员。港大的社堂，同样也有迎新营（Ocamp）等活动，有的社堂还保留了过Ocamp才吸收入内堂的文化传统，没过Ocamp只能当宿友（resident）而非舍友。但是美国兄弟会和港大社堂也有很大区别。 

[![](c612abb5-d8e3-4acb-a426-bf4a1cf9d682.jpg)](c612abb5-d8e3-4acb-a426-bf4a1cf9d682.jpg)

 第一，它是一个自愿参加的组织。在港大大陆生被要求在社堂度过第一年，没有选择余地。 第二，它很贵。我在UNC的宿舍旁就有一片兄弟会的住房。在同一片区，学校的宿费是每个月550-650美金，私营宿舍的宿费连部分餐费是900+美金，而兄弟会宿费连餐费是3000美金。为什么兄弟会这么贵？因为参加兄弟会，可以加入一个小圈子（coterie），这个小圈子内的人共享资源，互相帮助，无论是在学业上，还是在日后工作上，都会带来极大好处。这种加入小圈子的需求（在港大社堂也有人称为brotherhood或Strong bonding），是一种欲求，不是一种必须，按照市场运作的规律，理应付出更高的价钱去获得这样一种优势，而政府没有任何理由补贴它。香港补贴学生住房的模式，是补贴床位，但是床位和SA剔除堂友的“权力”联系在了一起，在实际运作中，SA往往以此为要挟，要求那些接受政府床位补贴的同学去参加社堂活动，否则quit hall。而一旦被quit hall，实际上就失去了政府住房补贴。所以，事实上SA在扭曲政策误导堂友，让大家以为政府补贴社堂文化。香港政府没有任何理由补贴社堂文化，就像美国政府没有任何理由补贴兄弟会一样。补贴社堂文化的是前堂友（alumni）。政府补贴的是住宿权力。而住宿权力与参加社堂文化的义务应该被切割开来，独立看待。 其次，从民主发展的角度来看，社堂制度是一个祸害无穷的误导者。 第一，SA是一个选举产生的机构。但是有选举，不代表有民主。民主的真正要义，是选民的意志能得到反映。香港社堂里选民的意志得到反映了吗？对于很多选民来说没有。许多选民并不想参加那么多社堂活动，这不但包括大陆生，也包括很多香港学生，甚至还有国际生。许多香港学生就是因为家住得远，迫于无奈才选择了便宜的社堂住。在社堂实际运作过程中，不管谁当选，都会要求堂友参加很多活动，对于那些不情愿的人，先是利用道德歧视，然后再辅以quit hall的威逼。在社堂里，有一种奇怪的意识形态，就是那些不参加活动，“不为社堂做贡献”的人，就没有资格住宿。在美国兄弟会里，你可以搞你的意识形态。在香港，也可以，比如像Dutchess这种已经没有住宿纠葛的纯“兄弟会”组织。但是在住宿空间里，你不可以。要搞，请到宿舍外面去。请不要混淆住宿和社堂文化两个概念。 第二，SA常常自以为自己是选举产生的组织，堂友们就有义务听从自己。实际情况是，很少有人愿意参选SA，所以经常看到十二人的内阁（cabinet）只有不到六个人参选，而且当选得票率很低。这样的组织，一点代表性都没有，自然也不能得到大多数人的认同。SA事实上没有什么强硬的执行力。假如你不服从，他们是没有办法的，只能到舍监耳边吹风去把你描述成一个不合格的堂友然后剔除。所以SA的权力来源主要是上文提到的两个，一是意识形态，二是谎称的quit hall的权力（事实上是建议quit hall的权力）。所谓意识形态，就是把不服从的人都抹黑成人民公敌，指责他们阻碍了社堂文化的伟大复兴，颠覆了乌托邦的社堂理想。然后对这些人冷眼相待。在这样的环境中生活是很不舒服的。虽然没有什么具体伤害，但是精神的压力也是一种伤害。这是威权制度的惯用伎俩。香港的繁荣来源于对个人自由的保护，而个人的自由包括不受歧视和言语侮辱或精神压力。社堂文化的实践，和自由民主的理想背道而驰。 第三，在这样的制度下实践的SA，甚至堂友们，很容易曲解民主。记得Morrison公开论坛上就有SA很委屈地说，我们是大家选出来的，难道不能代表大家做事么？也有堂友觉得，社堂民主的结果就是多数人的暴政，或者说是少数人以多数人的名义进行的暴政。事实就是，所谓“社堂民主”，根本不民主。民主的前提是对个人基本权利的保护。在美国的宿舍，如果有人半夜三更在楼道里大声喧哗，是要被处罚或者驱逐的。而在港大的某些社堂，SA还略带逼迫地鼓励开门政策（open door policy），更不用说那些半夜三更大喊大叫的人了。在港大社堂这样的环境中成长起来的香港人，能肩负起香港民主化的重任么？ 越来越合理的公共财政开支和正确的民主实践本身就是民主化进程的一部分。从这个角度来看，香港社堂制度的改革实在是意义重大。具体的做法其实有政府和堂友两个层面。政府层面（或学校层面），应该把补贴和床位分割。学生住房补贴直接发放到有需要的学生账户上，不管他是大陆生，香港生，还是国际生。相对应的，社堂宿位应该涨价，直到与合理的市场价格持平。这样想体验社堂文化的人继续住，不想体验的人就可以自由地找地方住。无论是SA还是堂友，这都是个皆大欢喜的结局。 

[![](a19ed972-aae5-4d88-9e69-0bac947c242c.jpg)](a19ed972-aae5-4d88-9e69-0bac947c242c.jpg)

 堂友层面，大家应该明白自己的权利所在。有很多不合理的做法，假如没有人反抗，就会约定俗成。中国人习惯性地指望有圣明的外部力量来替自己主持公道，却很少尝试伸张自己的利益。对不合理现象的步步紧逼一味忍让。须知，即使在自由民主的美国，假如你不出声，是没有人来保护你的权益的。你必须拿出勇气来正视现实。假如你觉得社堂生活很愉快，那么恭喜你。但是请明白，这样一个社堂制度在给你带来愉悦的同时给另外一些人带来了痛苦。假如你觉得社堂生活不愉快，但是甘愿忍受，那么这是你自己的选择。 假如你觉得社堂生活不愉快，而且无法忍受，那么你可以选择搬走，或者出声反抗。如果你什么都不愿意做，那么恕我直言，你活该。 至少在香港，是有条件有自由去伸张自己的权力的。中国人如果不明白去为自己的权益奋斗，而一味指望圣主明君来为自己主持公道的话，就活该被专制。
