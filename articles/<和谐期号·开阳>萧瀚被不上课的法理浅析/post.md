# 萧瀚被不上课的法理浅析

## 文 / 陈尉 ( 中南政法大学 )

谈起萧瀚先生我总有几分自豪，因为我与他有过一面之缘，那次是在珞珈山上聆听了他的教诲，他可以说是我私下见过的第一位大师了（第二位是邓晓芒），所以至今我记忆犹新。我也读过先生的书和很多文章，我被他渊博的知识和胸襟深深吸引，我觉得他是我国少有的真的教师1和知识分子，他不仅拥有知识，而且作为一个知识人他配得上这些知识。 昨日，突然看到一则新闻“青年学者萧瀚“被不上课”或因介入公共事务太深”——萧瀚先生被不上课了，我并不震惊，因为这在中国并不是一个骇人听闻的事情，只是一件小事而已。这件事再次印证了有关部门是我国最大权力部门的坊间传言，同时也体现了有关部门对我国大学的绝对领导。这是一个怎样的国家和怎样的政府我并不关心，毋宁说是不敢关心，其实每个人都心知肚明，在此不作探讨。我无意在此美化萧瀚而大发牢骚，因为那毫无用处且显得幼稚，我将从一个人、公民、教师和劳动者的角度分析这个事件，以法理服人。 

### 1.正当法律程序的拷问：

程序是指人们遵循法定的时限和时序并按照法定的方式和关系进行法律行为2，程序是现代文明社会的根本标志。就正当程序的最低标准而言，它要求：公民的权利义务将因为决定而受到影响时，在决定之前必须给与他知情和申辩的机会和权利。对于决定者而言，就是履行告知（notice）和听证（hearing）的义务。 就此事萧瀚在博客中写道，“并没有人来通知我，你被解雇了，或是你被停课，而是在下学期的本科生，研究生的选课表上，没有了我的名字。”这说明在被停课之前，萧瀚没有得到任何消息，也可以看出，这次停课是早有预谋的。 而萧瀚先生最近的博文又说道，“其实，到现在我也不知道到底是什么原因，什么程序，什么理由，以及什么人作出这个停课决定。报道上说停课是“或因介入公共事务太深”，是记者问到这一点的时候，我说可能吧，并非我自己这么认为。”在此我们能够得到一些什么信息呢？那就是停课决定早就有预谋的下达给了校方，而校方也一直在秘密地执行着，不论是有关部门还是校方，都没有提前告知萧瀚先生，也没有人事后来宣称对此事负责，萧瀚先生现在是事实上被不上课了，但是没有任何正式的文件或消息来说明这件事，这是谁的决定、为什么要停课等等。 以上种种已经证明，对萧瀚先生的停课不符合程序、法律程序和正当的法律程序，这个决定是完全的暗箱操作，于礼无据、于法无依的决定怎么会在现代文明社会形成、作出并生效，我百思不得其解。（也许自古以来的文明古国本身并不文明） 

### 2.管辖权问题

我们必须追问，把萧瀚先生停课是谁做出的决定，他有这个权力吗？这就涉及到一个管辖权的问题。 这里我们必须分析萧瀚先生的角色：作为一个教师，也就是为学校工作的劳动者，教育方面的问题归学校管辖，因此对他在教学方面的决定理应并只应由学校作出，因为这涉及劳动双方的权利和义务，只有劳动者和用人单位才有权去决定，校方也一直宣称，萧瀚是一位好教员。同时，萧瀚先生也是一位公民，因此他的行为都要受到法律的约束，他的一切行为都要受到法律的审阅，法律将制裁他的违法犯罪行为。但是我们可以看到，萧瀚先生是一位合法的公民，他没有受到审判，没有被判处有罪，因此萧瀚先生不可能因为是公民而招致惩罚。但是，萧瀚先生且被停课了，这就很不可思议了：作为一名用人单位赞誉、被服务者喜欢的劳动者，作为一位合法的公民，就这样被停课了。怎么会这样呢？ 对萧瀚先生有管辖权的学校和公权力部门都没有对他进行管辖和制裁，但事实上他却被严厉地制裁了，难道还有第三只看不见的手在“掌管”着萧瀚先生吗？我不敢往下想。 

### 3.劳动者萧瀚：

前面说过，萧瀚先生是一位劳动者，作为一名职业教师，教书是他谋生的主要手段、是他追求幸福的基础，他靠在大学教书取得收入，实现自己的价值；然后现在他却“被不劳动”了，他没有被解雇，没有被解职，但是他却被不上课，这是对劳动者劳动权的严重侵犯。 世界人权宣言第二十三条规定：“人人有权工作、自由选择职业、享受公正和合适的工作条件并享受免于失业的保障”。萧瀚先生现在还是一名劳动者，但是却丧失了工作的权利，他在事实上失业了，那么他将靠什么谋生呢？他靠什么去维持最基本的生活、他靠什么去追求更高的生活？没有工作的人就像折了翅膀的雄鹰、断了线的风筝，最后都将陷入困境，这难道不是人道主义灾难吗？这难道不是对人、对人权赤裸裸、血淋淋的侵犯吗？ 

### 4.他去哪里寻求救济？

不管是“无救济即无权利”还是“无权利即无救济”，我们都认为权利被侵犯了我们有寻求救济的权利，然而萧瀚先生却要面临无法救济的困境。被停课的决定是谁作出的仍然是个谜，所以萧瀚先生将无法寻求救济，因为这里有两个问题需要解决：首先，向谁寻求救济？其次，你的相对人（即作出决定的人）是谁？ 我们知道，无论是申诉和诉讼，都必须是具体的侵权事实，特别是要有明确的主体，而在这件事中，作出决定的人不是学校也不是国家审判机关这样对萧瀚有管辖权的机关，而是看不见的第三只手，这就造成了萧瀚被侵权的另一方主体的缺失，这样就会使得萧瀚先生无法去寻求救济，因为他不知道谁侵了他的权。 其实第二个问题相对简单：萧瀚先生可以去向谁寻求救济呢？首先学校接了通知而执行了，这说明向学校寻求救济不可能，学校已经执行了这个比他权力大的机关的决定，显然学校已经没有审查这个决定的权力和勇气。其次，如果提起诉讼，一来被告是谁都难以确定，二来找不到法院受理。因为如果提起诉讼，那么更类似行政诉讼，但是萧瀚先生却难以找到具体的侵权行政行为；而法院可能把此认定为一个劳动争议，因为萧瀚先生丧失的从根本上来说是劳动权，但事实上他与学校都很满意彼此。其实，法院根本不会受理萧瀚先生的诉求，因为法院也惧怕那看不见的第三只手，一旦进行诉讼，萧瀚先生必然会像三鹿受害儿童的家长一样，诉讼无门。 总之，这是一个无法救济的决定，这是一个不允许萧瀚去寻求救济、不允许任何人、社会团体或机关去救济萧瀚先生的决定。 

### 5.结语：被时代才刚刚开始

周永坤先生说过，“被”时代其实是人治时代的特色，“被”表明人没有自主，人不能把握自己的命运，只能任由外在力量摆布，这正是所有人治社会的特色。人如何才能自主，因而才能摆脱“被”的命运？相信唯一的办法就是实行法治。4被死亡、被和谐、被不上课···，被时代是中国特色的又一个产物，必将彪炳历史。 长太息以掩涕兮，哀民生之多坚，让我们以不死之心迎接被时代的到来吧！
