# 诗情也似并刀快，剪得秋光入春来——读《我与地坛》

# 北辰“灯下漫笔”专栏开栏说明

笔者自幼喜读史书，长于杂文和议论，以行文流畅，事例充实，语言直白为追求。曾于天玑板块开辟专栏指点江山，漫谈史话，自以为得。至近期博览众多巨匠之著，方悟己先前之浅薄。惭愧之余，乃停笔读书一月有余。培根曾经说过：“读史使人明智，读诗使人聪慧，学习数学使人精密，物理学使人深刻，伦理学使人高尚，逻辑修辞使人善辩。”史书对于增长见识，开阔眼界大有裨益；然而唯有文学才能滋养出高贵的心灵。 诗云“试玉要烧三日满，辨材须待七年期”。只有读书，方可开阔眼界，增长智慧，启迪心智，升华灵魂。在本专栏中，笔者将以读书笔记的形式，和众位读者一起共同感悟书中的哲理，和伟大的作者们一道追忆似水年华，遨游精神乐园。 

# 诗情也似并刀快，剪得秋光入春来

# ——读《我与地坛》

他们要用手托着你， 免得你的脚碰在石头上。 ——《圣经.诗篇91：12》

这是一篇与灵魂交流的文字。 不知为何，总是被这篇散文中的一些看似不起眼的词句打动。总是不自觉的和作者共同思考人生命运的问题。在这有如潺潺流水般的文字当中，我读出了有关时间及人生意义最确切的答案。 地坛无疑和作者有着某种缘分，在某种程度上，地坛和作者也有着许多共性：同样是被人所遗忘；同样是没有融入这巨大的、复杂的、日新月异的现代社会。因而作者选择了地坛，正如地坛选择了他一样。在这荒芜而又寂静的古园里，作者寻找着自己对生命的感悟，写下了一篇篇叩响心灵的文字。这深思凝练成的文字，让作者的灵魂与古老静谧的地坛融为一体。 作者虽然在最狂妄的年龄上残废了双腿，饱尝命运的不公。但是，他还是一个幸运的人，不为别的，就为他有一位伟大的母亲。母亲将她全部的生命献给了自己的儿子，她理解自己的儿子，知道他需要一个宁静的环境进行思考，但又害怕他不能释怀，会因双腿的残疾失去对生活的信心。同时，母亲又非常希望能与儿子沟通，替他分担痛苦，让他发泄情感。她祈祷着自己的儿子能拥有丰富，充实的人生，盼望着他能找到属于自己的幸福。女性的本性是纯洁的，善良的，女性是为了爱而生的。母亲的爱如一池的云影，一天的涛声，一舟的明月，一袭盈袖的暗香。这样的爱，使我们有勇气和信心在黑暗中寻找光明，在痛苦中追逐欢乐，在贫瘠中品尝丰收，在屈辱中捍卫荣誉。正如《圣经.雅歌》中所说：“才德的妇人，谁能得着呢？她的价值远胜过珍珠。” 然而，当作者越来越深刻的体会到母亲无私的关爱和内心的矛盾痛苦时，作者再也无法为这位伟大的母亲做些什么了。 我以前从未如此认真的思考过生与死的问题，因为这些问题对于涉世尚浅的我来说，显得过于沉重了。读罢此文后，我忽然意识到这其实是对生命的尊重与责任。一直以来，人类总是被一个问题所困扰：我们为什么不能坦然自若的面对死亡？一位美国的社会学学者说过，最根本的原因在于：人没有把自己看作是自然的一部分。我们不会在意一块石头的碎裂，一棵树木的枯荣。因为它们的生命，不能被我们放进四季的时序中去考察。木石的生命存在于悠远无垠的时间之外，几乎就是时间的一部分。人类无法占有和支配时间，所以，人类永远克服不了生命的有限性。同样的，一片树叶落下了，我们不会悲伤，因为它是自然的一部分；一朵花枯萎了，我们也不会悲伤，因为它是自然的一部分。如果人能够把自己看成是自然的一部分，那么他就不会耽于生而惧于死了。因为生是自然的一部分，死也是自然的一部分。十五年来，作者与地坛朝夕相处，已经把自己的生命融入了自然中。从这点来看，作者的精神是富有的，相比很多身体健全的人，他也是幸运的。 生，是上帝交给的事实，也是母亲的创造。一个人既然获得了生命，就有活下去的权利，这其实也是每个人对于生命的责任，你必须用生命去体味生活，感悟生活。你无法决定命运中美好或者悲惨的偶然性——那是上帝的安排，但你可以凭自己的信念去努力生活，可以在饱经痛苦的同时用心去体味世间万物的美好，让你的生变得有意义，其实这就是幸福。这与命运的抗争不正是在实现着生活的愿望吗？只不过每个人生活的方式不同，作家的方式是在不停地写作，而我们的方式又是什么呢？这的确需要时间来思考清楚。对于死，从生的那天开始就是注定了的，是上帝自然的安排，无需悲哀，无需恐慌，上帝总有一天会召唤你回去。现在，我只希望当那一天到来之时，回顾我在自然中短暂的生命，会由衷的感谢上帝，赞美上帝，感激母亲赐予了我生命，然后从容的和这个世界告别。 又是一个十月，地坛的落叶随着萧瑟的秋风，旋转纷飞在如洗的天空中，如山花般盛开，如清泉般流泻，如时光般永久，如生命般高贵。寻觅与归宿，时间与空间，生存与死亡，这些永恒的话题在破败的地坛里，在作者的脑海中汇聚成一句话：愿我们都能好好地活着。
