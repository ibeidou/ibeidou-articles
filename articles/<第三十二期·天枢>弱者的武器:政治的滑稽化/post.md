# 弱者的武器:政治的滑稽化

# 弱者的武器:政治的滑稽化

## 文／中国政法大学学生

不知道从什么时候开始，网络上的各种搞笑的无厘头式的流行词越来越多，像周老虎，打酱油，范跑跑，俯卧撑，草泥马，河蟹等等。这些流行词个个短小精悍，朗朗上口，表面看来似乎并不是晦涩难懂，但是每个词都有着具体的事件语境和深刻的社会内涵，对于那些不明白具体语境的人来说，理解这些词语的难度将比理解世界上任何一种理论都要高，说白了，根本无法理解。 这些词语逐渐演变成为我们生活中的工具和思维，而且每时每刻都在不断加入新的成员，比如最近又出现了一个叫躲猫猫。它们的意思差别很大，但都表达了超越名词本身的的含义，具有更深一层的社会内涵。它们是词语，但更是一种当下中国人的生存状态。 尽管这些词语间有着千差万别的含义，但同时他们又具有很多的共同之处：这些词语本身都非常搞笑、滑稽，具有很强的无厘头性，结构凝练简洁，读起来上口，而且根本不是正常的汉语表达方式——这是它们在形式上的共同之处。但这种共性并不重要，这些词之所以这么为大众所喜爱，源于另外一个本质性的共同之处，那就是这些词语所阐述的具体的内涵，所表达的社会现象，所发出的社会呼声。这些词语本质上已经不是一种“词语”，而是社会大众在这个专制体制下变相的表达方式。 这种表达方式非常新颖，但它的精神核心却由来已久，可以说是伴随了人类反对专制统治的整个过程，大致追踪的话也有几千年的历史。这种表达方式是什么呢？用美国一位学者的定义就是——弱者的武器，和隐藏的文本。 美国著名的政治人类学家詹姆斯•斯科特在对东南亚农民的反抗实践进行民族志调查研究的基础上，提出了“弱者的武器”和“隐藏的文本”两个重要概念，用以解释底层群体生存与反抗的逻辑。所谓“弱者的武器”指的是农民反抗的日常形式，即农民同那些索取者、压迫者之间平凡的却持续不断的斗争。这些日常形式的反抗通常包括：偷懒，装糊涂，开小差，假装顺从，偷盗，装傻卖呆，诽谤，纵火，怠工等等。这些被称为“弱者的武器”的阶级斗争形式具有共同特点：它们几乎不需要事先的协调和计划，而是利用心照不宣的理解和非正式的社会网络，通常体现为一种个体的自助形式；它们避免直接地、象征性地对抗权威。这类几乎不着痕迹的“弱者武器”的运用是由于公开的、有组织的政治行动对于多数下层阶级来说是过于奢侈了，那即使不是自取灭亡，也是过于危险的。 “隐藏的文本”指的是相对于“公开的文本”而存在的、发生在后台的话语、姿态和实践，它们避开掌权者直接的监视、抵触或改变着“公开的文本”所表现的内容。斯科特指出，每一从属群体因其苦难都会创造出“隐藏的文本”，它表现为一种在统治者背后说出的对于权力的批评，它使从属者可能破除“虚假意识”和神秘化的迷障。作为底层政治的“隐藏的文本”，有助于理解底层群体安全地表达自己的对社会的不满。 之所以会出现这种弱者的武器，隐藏的文本，根本原因就在于在这么一个专制的政治体制下，我们都是弱者，我们没有强有的力量和统治者抗争，我们不敢也没有渠道使用公开的话语和他们抗争。所以我们只能使用这种消极的对抗方式，使用这种隐藏的文本，这就是我们这些弱者的武器。对于这种隐藏的文本，专制者根本防不胜防，无能为力。 这是一种消极的表达方式。我们都明白，在这种专制体制残酷的统治下生活，我们根本无法用积极和公开的的抗争方式和他们斗争，争取我们的权利。我们被无情地剥夺了在公开场合表达自我的机会和自由，现在的报纸，电视，乃至网络都成为了官方的喉舌，我们的思想自由、言论自由、出版自由、集会自由、游行示威的自由等等，我们很多很多的自由，都被这个专制的政府剥夺的干干净净。在这个我们每天为之贡献爱心、智慧和力量的社会，在这个对我们的切身利益影响巨大的社会，渺小的我们失去了表达自我、伸张自我和保护自我的权利和能力，我们每个人在专制的权力面前都成了一只可怜的脆弱的羔羊。 可是，我们毕竟不是羔羊，我们是人，是一个具有灵魂、人格、尊严，是一个具有欲望和追求的人，所以我们必然不会彻底地默默忍受这种非人的体制和生活状态，我们必然要找到一种新的表达方式，我们必须有所表达，而隐藏的文本就是我们这种状态下的表达方式。但这种隐藏的表达方式和公开的表达方式有着本质性的不同，因为隐藏的表达方式本质上是消极的，被动的，并不能对专制者有直接的冲击，对我们被损害的权利和利益没有多么大的作用，更不能有效表达我们的呼声和维护我们的家园，它仅仅是一种消极的表达，无奈的抗议。我们也没有奢望这种表达能对权利和利益的维护起到多么实际的作用，而是通过戏虐的方式，强烈地讽刺和抗议专制的体制和邪恶的社会。 这就是弱者的武器，这就是隐藏的文本，这就是我们大多数人在这个专制残暴的体制下无可奈何的表达方式，这种表达方式表面上看来轻松活泼，大家都在当作玩笑一样地随口说出。比如：我是来打酱油的，我在做俯卧撑，你今天躲猫猫了吗？我是一只草泥马。等等，已经成为了很多人的口头禅，但是在这种戏说性表达方式的背后，却蕴含了我们每个人巨大的无奈和悲哀。这是一种带着眼泪的欢笑，这是一种强作欢颜的悲哀。在这种表达的背后，暴露了在这个专制体制下每个卑微草民的悲哀的生活状态；暴露了我们每个人对这个专制体制的愤怒和无可奈何；表明了我们对专制的政治体制强烈的敌视和不满。弱者的武器，更多的不是武器，而是说明了我们是弱者。 并且，这种武器的流行也说明了中国人的性格懦弱和卑微，我们大多数人不敢追求自己的权利，不敢坚持自己的人格，在现实面前，大多数人不得不放弃自己的宝贵的灵魂和尊严，甘愿成为没有人格的奴才，只能在网络上发泄和抗议，很多人成为了双重人格，活在灵魂的分裂之中。这种武器正是中国人懦弱的性格的真实写照，正是中华民族最卑劣的特性的集中反映，这就是外国人看不起我们的根本原因：中国人最缺乏的不是智慧，而是勇气和正直的纯正品性。 但是，不论怎样，这是我们中国人目前作出的抗争方式，依然具有它自身的价值和意义，所以我们绝对不能轻视这种武器，不能低估它的能量，虽然这种武器并不强大，但它却是一个具有历史意义的启示。因为正是通过这种方式，我们表明了这样一种国民心理和社会状态，那就是我们不再是那种完全逆来顺受、听天由命的奴隶、不再是没有任何反思和批判能力的顺民、不再是专制体制下噤若寒蝉的草民。不，我们不再是这样了，我们开始觉醒，我们开始抵制，我们开始抗争。我们虽然不能公开地质疑专制者，但我们在思想上却不再是专制体制的逆来顺受的羔羊；我们虽然不敢公开抵制专制者，但我们能够对他们的一切话语和行动都不再信任，不再尊敬，我们的心灵开始获得了独立的自由的空间，再也不会被专制者轻易地欺骗和愚弄。 当专制者不再被尊重，当专制政治越来越无法得到每个人的发自内心的关心和尊重，政治在人们心目中就像是一群小丑的儿戏，一群丑陋的群魔乱舞，政治越来越同人民相隔膜，同时也没有几个人关心这样的政治。政治仿佛和我们毫无关系，仿佛是我们生活中的多余之物，我们不再严肃地看待政治。 一个社会沦落到这样的心理和思维状态，最终将会产生对专制者致命性的打击，哀莫大于心死，当我们每个人对政治都漠不关心，毫无信心的时候，这种政治体制的死期也就快到了。尽管这种打击是和风细雨和潜移默化式的，但是，在这种对政治的讽刺和戏虐的心理状态下，专制政治从此会逐渐丧失人们的认同，专制者的政治合法性也会逐渐地丧失，最终为根本性的政治变革奠定不断壮大和深厚的群众基础。 这就是弱者的武器，这就是隐藏的文本，这种武器的出现要感谢网络的流行和不断普及，没有网络，就没有这种流行词的生存空间。幸运的是，我们拥有了网络。毋庸置疑，在这样一个网络时代，这种弱者的武器和隐藏的文本，会像雨后春笋般地不断出现，它的繁殖速度将会大大超出每个人的预料之外，官方的各种言语、行为，丑闻，民间的各种言语、行为、活动，等等，都随时有可能成为一个个新的流行词语。现在，如果谁10天左右不上网络，就非常有可能落伍，发现一些他不懂的新词。 在一个专制的时代，在一个无权的时代，在一个我们都是弱者的时代，这就是我们抗争的武器，这就是我们的非暴力不合作运动，这就是我们隐藏的文本和弱者的武器： 专制统治者想要河蟹，可我们这些国家的公民偏偏要草泥马。
